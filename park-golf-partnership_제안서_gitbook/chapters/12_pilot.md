# 4.3 파일럿(시범 운영) 세부 계획

---

## 4.3.1 파일럿 대상 선정 기준

아임인 파크골프 파트너십의 첫 파일럿 매장은 아래 기준을 충족하는 대형 스크린 파크골프장을 우선 선정합니다.

| 기준 | 세부 조건 |
|------|---------|
| **매장 규모** | 150~200평 이상 (10타석 이상) |
| **고객 기반** | 기존 동호회 회원 또는 정기 방문 고객 30명 이상 보유 |
| **운영 의지** | 대표 또는 지점장의 적극적 참여 의사 확인 |
| **지역 위치** | 인근 경쟁 파크골프장 대비 집객력 확대 필요성이 높은 곳 |
| **직원 역량** | O2O 라운지 담당 직원 1명 이상 지정 가능 |

---

## 4.3.2 파일럿 운영 프로세스

### Step 1. 킥오프 (D-Day)
- 계약 체결 및 파트너십 공식 발표
- 매장 내 아임인 홍보물(포스터·배너·테이블 스탠드) 설치
- 직원 교육 (약 2시간): 아임인 서비스 소개 + O2O 가입 프로세스 실습

### Step 2. O2O 라운지 가동 (1개월차~)
```
일별 운영 루틴

  오전 오픈 시
  └─ 입구 아임인 홍보 배너 확인 및 리플릿 보충

  라운드 피크 타임 (오전 10시~오후 3시)
  └─ 담당 직원이 휴게 공간에서 방문객과 대화
     "아임인 스테이지 알고 계세요? 여기서 가입하면 이용권 드려요"

  라운드 후 휴식 시간
  └─ 관심 있는 방문객 대상 1:1 가입 지원 (15분 내 완료)

  일 마감 시
  └─ 당일 가입자 수 티웨이브 담당 매니저에게 보고
```

### Step 3. 파일럿 대회 개최 (2개월차)
- **대회 규모**: 참가자 50-100명, 총상금 500만-1,000만 원
- **참가 조건**: 아임인 스테이지 신규 가입 또는 기존 참여자
- **진행 방식**: 매장 기존 운영 방식 그대로 (아임인은 상금·경품만 지원)
- **홍보**: 동호회 단체 카카오톡, 매장 SNS, 아임인 앱 내 공지

### Step 4. 홀인원 스폰서십 가동 (1개월차~)
- 매장 내 "아임인 홀인원 스폰서십" 안내문 부착
- 홀인원 발생 시 → 담당 직원이 아임인 측에 즉시 보고 → **24시간 내 현금 30~50만 원 지급**
- SNS 인증샷 이벤트 병행 ("홀인원하고 현금 받았어요!" 후기 유도)

## 4.3.3 KPI 측정 방법

| KPI 항목 | 측정 방법 | 측정 주기 |
|---------|---------|---------|
| 스테이지 신규 가입자 수 | 아임인 플랫폼 내 제휴 코드 기반 추적 | 주 1회 |
| O2O 라운지 가입 전환율 | (가입 완료 / 상담 진행) × 100 | 월 1회 |
| 매장 타석 가동률 변화 | 파일럿 전후 일 평균 타석 이용 시간 비교 | 월 1회 |
| 재방문율 변화 | 아임인 가입 고객의 월 평균 방문 횟수 | 월 1회 |
| 홀인원 발생 건수 | 현장 직원 보고 | 건별 즉시 |
| 인센티브 지급액 | 아임인 정산 시스템 | 월 1회 |

## 4.3.4 파일럿 성공 기준

3개월 파일럿 종료 후 아래 기준 충족 시 → Phase 3(확대) 자동 전환

```
✅ 스테이지 누적 가입자 30명 이상
✅ O2O 라운지 월 가입 건수 10건 이상 달성
✅ 매장 재방문율 기준치 대비 +20% 이상
✅ 파일럿 대회 참가자 만족도 80% 이상
✅ 매장 담당자의 지속 운영 의사 확인
```
