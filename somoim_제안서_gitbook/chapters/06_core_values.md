## 2.2.1 아임인 3축 가치와 소모임 적합성

아임인은 세 가지 핵심 가치를 통해 소모임 유저들의 공동 자금 수요를 완벽히 충족합니다.

## 2.2.2 가치 축 1: 안전성 — 원금 구조적 보호

소모임 유저들이 공동으로 돈을 모을 때 가장 걱정하는 것은 '내 돈이 안전한가'입니다. 아임인은 3중 안전장치로 이를 구조적으로 해결합니다.

| 안전장치 | 내용 | 소모임 적용 의미 |
| -------- | ---- | ---------------- |
| **KCB 신용 확인** | 참여 전 신용 스크리닝으로 그룹 내 디폴트 리스크 최소화 | 모르는 유저와 함께해도 안전 |
| **안심서비스** | 연체 발생 시 보험사 대납 → 다른 멤버 원금 100% 보호 | 한 명 미납해도 나머지 영향 없음 |
| **제1금융권 에스크로** | 신한은행·케이뱅크 연계 에스크로 계좌 분리 보관 | 티웨이브 파산 시에도 자금 보호 |

**결과:** 8년 운영 기간 중 연체율 0.22% — COVID 팬데믹(2020~2021), 고금리 시기(2022)에도 1% 미만 유지.

## 2.2.3 가치 축 2: 유동성 — 소모임 활동 주기 맞춤 설계

소모임마다 목적 자금이 필요한 시점이 다릅니다. 아임인은 다양한 스테이지 기간으로 이를 지원합니다.

| 스테이지 | 소모임 활용 예시 |
| :------: | ---------------- |
| **5개월** | 6개월 후 다가오는 소모임 여름 여행 자금 |
| **7개월** | 연말 행사·송년회 예산 마련 |
| **9개월** | 내년 봄 등산 장비 업그레이드 자금 |
| **11개월** | 연 1회 해외 여행 시드머니 |
| **13개월** | 2년 주기 대형 행사 적립금 |

> 소모임의 연간 활동 계획에 맞춰 스테이지를 선택하면, 행사 시점에 맞게 목돈을 수령할 수 있습니다.

## 2.2.4 가치 축 3: 소셜 동기부여 — 함께여야 지속된다

아임인의 가장 강력한 차별점은 '혼자가 아닌 함께'라는 소셜 메커니즘입니다. 이는 소모임의 핵심 가치와 완벽하게 정합합니다.

```
소셜 동기부여 메커니즘:

① 그룹 단위 가입     — 혼자가 아닌 소모임 멤버들과 함께
② 진행 현황 가시화   — 스테이지 진행 상황을 앱에서 실시간 확인
③ 공동 성취감        — 목돈 수령의 기쁨을 함께 나눔
④ 선임-후임 전파     — 성공한 멤버가 새 멤버 자연스럽게 초대
```

> 혼자 저축할 때는 쉽게 포기합니다. 하지만 소모임 멤버들과 함께 모을 때는 '나 하나 빠지면 모임에 민폐'라는 사회적 책임감이 저축 완주율을 높입니다. 아임인의 연체율 0.22%는 이 소셜 메커니즘의 효과를 수치로 증명합니다.
