## 3.1.1 아임인(imin)이란?

아임인은 한국의 전통적인 '계(契)' 문화를 디지털로 혁신한 소셜 저축 금융 플랫폼입니다. '계'는 수백 년 역사를 가진 상호 부조 금융 방식으로, 구성원이 일정 금액을 정기적으로 납입하고 차례대로 목돈을 수령하는 구조입니다. 아임인은 이 원리에 디지털 기술, 신용 평가, 보험 연계를 결합하여 현대적으로 진화시켰습니다.

## 3.1.2 스테이지 운용 원리

### 기본 구조 (9인 그룹 예시)

```
┌──────────────────────────────────────────────────────────┐
│  9명이 한 그룹 → 매월 동일 금액 납입 → 순번대로 목돈 수령  │
└──────────────────────────────────────────────────────────┘

예시: 9명 × 월 30만 원 × 9개월 스테이지

 1개월: A 수령 → 270만 원 (9명 × 30만 원)
 2개월: B 수령 → 270만 원
 3개월: C 수령 → 270만 원
 ...
 9개월: I 수령 → 270만 원

모든 멤버가 동일하게 270만 원을 수령하고 스테이지 종료.
```

### 소모임 전용 스테이지: 3~9인 소그룹 운영 가능

일반 아임인 스테이지(9인)와 달리, 소모임 전용 스테이지는 소모임 구성원 수에 맞게 **3~9인**으로 유연하게 운영됩니다.

## 3.1.3 스테이지 운영 프로세스

| 단계 | 내용 |
| :--: | ---- |
| **① 목표 설정** | 소모임 내에서 자금 목표 합의 (장비 구매, 여행, 행사 등) |
| **② 스테이지 선택** | 5·7·9·11·13개월 중 행사 일정에 맞게 선택 |
| **③ 그룹 초대** | 소모임장이 멤버를 아임인으로 초대 (링크 공유) |
| **④ KCB 확인** | 참여자 신용 정보 자동 확인 → 순번 배정 |
| **⑤ 자동 납입** | 매월 지정일 자동 이체 — 급여일 연동 설정 가능 |
| **⑥ 목돈 수령** | 순번에 맞춰 목돈 일시 수령 (전 멤버 해당) |
| **⑦ 지원금 지급** | 스테이지 완료 시 티웨이브가 1인당 5~7만 원 활동 지원금 추가 지급 |

## 3.1.4 주요 기능 5가지

| 기능 | 내용 |
| ---- | ---- |
| **자동 매칭** | 소모임 멤버 초대 없이도 플랫폼이 자동으로 그룹 완성 |
| **실시간 현황** | 스테이지 납입 현황·잔여 기간·예상 수령일 앱 내 실시간 확인 |
| **자동 납입** | 등록 계좌에서 자동 이체 — 깜빡 잊음 없음 |
| **안심서비스** | 미납 발생 시 보험사 대납 → 다른 멤버 영향 없음 |
| **소셜 커뮤니티** | 출석 체크, 댓글, 응원 메시지 — 소모임 내 동기부여 강화 |
