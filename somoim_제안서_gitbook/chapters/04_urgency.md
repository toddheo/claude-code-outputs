# 1.4 도입의 긴급성: 지금이 골든타임인 이유

**[2026-05-12]**

---

## 1.4.1 커뮤니티 플랫폼의 금융화는 피할 수 없는 흐름

전 세계적으로 커뮤니티 플랫폼과 금융 서비스의 결합은 이미 대세가 되었습니다.

| 사례 | 내용 |
| ---- | ---- |
| **당근마켓** | 당근페이 론칭 — 지역 커뮤니티에 결제·송금 기능 통합 |
| **네이버 밴드** | 밴드 내 공동구매·결제 기능 실험적 도입 |
| **글로벌 ROSCA 플랫폼** | Money Fellows(이집트, $60M+ 투자 유치), Mapan(인도네시아) 등 공동 저축 플랫폼이 글로벌 VC 투자 흡수 |

소모임 플랫폼에 금융 기능을 통합하는 것은 '할 것인가'의 문제가 아니라 **'언제, 누구와 할 것인가'**의 문제입니다.

---

## 1.4.2 경쟁 플랫폼의 선점 위험

소모임과 유사한 포지션을 갖는 커뮤니티 앱들이 금융 파트너십을 검토하는 흐름이 가속화되고 있습니다. 프렌즈큐브가 먼저 파트너십을 구축하면:

```
소모임 × 아임인 = 국내 최초 커뮤니티 기반 공동 금융 플랫폼
```

이 선점 효과는 언론 보도, 유저 바이럴, 경쟁사 진입 장벽으로 이어집니다. 반면 도입이 늦어질수록 경쟁 커뮤니티 앱이 먼저 유사 파트너십을 구축할 위험이 있습니다.

---

## 1.4.3 소모임 유저의 '돈 모으는 시즌' 실기 위험

소모임 활동은 계절성이 강합니다. 지금 파트너십을 구축하면:

| 시점 | 활용 가능 시나리오 |
| ---- | ----------------- |
| **2026 하반기 론칭** | 2026 연말 행사 자금 마련 챌린지 (가장 강력한 첫 캠페인) |
| **2027 상반기** | 봄 여행·등산 시즌 공동 자금 챌린지 |
| **2027 연중** | 소모임 연간 활동 주기와 완전히 연동된 상시 운영 |

2026년 하반기 론칭을 놓치면, 연말 행사 시즌이라는 가장 강력한 초기 캠페인 기회를 잃습니다.
