# 1.3 현행 대안의 구조적 한계

**[2026-05-12]**

---

## 1.3.1 현행 대안 분석

소모임 유저들이 현재 목적 자금 마련을 위해 사용하는 수단과 그 한계를 분석합니다.

| 현행 대안 | 핵심 한계 | 소모임 적합성 |
| --------- | --------- | :-----------: |
| **카카오페이 더치페이·송금** | 즉시 분담은 가능하나, 목표를 향해 '함께 모으기'는 불가. 미래 계획 불가. | ✗ |
| **모임장 개인 계좌 취합** | 신뢰 의존. 횡령·분실 리스크. 정산 분쟁 시 중재 불가. | ✗ |
| **일반 개인 적금** | 공동 참여 구조 없음. 소셜 동기부여 없음. 목표 공유 불가. | ✗ |
| **크라우드펀딩 플랫폼** | B2C 프로젝트 성격. 소규모 소모임 자금 마련 목적 부적합. 수수료 과다. | ✗ |
| **아임인 소모임 전용 스테이지** | 3~9인 공동 참여. 목적 자금 공유. 완료 시 티웨이브 지원금. | ✓ |

---

## 1.3.2 한계 1: 카카오페이·N빵의 '분담은 가능, 적립은 불가' 구조

카카오페이 더치페이와 N빵 기능은 **이미 발생한 비용을 나누는** 데 최적화되어 있습니다. 그러나 소모임 유저들에게 필요한 것은 **미래의 목표를 향해 함께 모으는** 기능입니다.

```
더치페이 모델:    지출 발생 → 사후 분담 (후불 정산)
아임인 모델:      목표 설정 → 함께 적립 → 목돈 수령 (선불 적립)
```

이 차이는 사용자 경험에서 근본적으로 다릅니다. 더치페이는 결제 후 청구 피로감을 유발하고, 미납자가 생기면 갈등이 발생합니다. 아임인은 스테이지 참여 시점부터 자동 납입으로 운영되어 갈등 요소 자체를 제거합니다.

---

## 1.3.3 한계 2: 개인 계좌 취합의 신뢰 의존 구조

모임장이 개인 계좌로 회비를 모으는 방식은 여전히 가장 흔하게 사용되지만, 구조적 취약성을 가집니다.

| 위험 요소 | 내용 |
| --------- | ---- |
| **횡령·분실 리스크** | 모임장 개인 계좌에 공동 자금이 혼재 |
| **정산 투명성 부재** | 잔액 확인 불가, 사용 내역 공유 어려움 |
| **법적 책임 소재 불분명** | 분쟁 발생 시 민사 소송 외 구제 수단 없음 |
| **모임장 부재 시 자금 동결** | 모임장 탈퇴·연락 두절 시 자금 접근 불가 |

소모임 커뮤니티 내에서 '돈 문제로 모임이 깨졌다'는 사례는 드물지 않습니다. 이는 개인 신뢰에 의존하는 현행 구조의 필연적 결과입니다.

---

## 1.3.4 한계 3: 일반 적금의 소셜 기능 부재

개인이 각자 적금을 가입하는 방식은 원금 보호 측면에서 안전하지만, 소모임의 '함께 한다'는 가치와 전혀 연결되지 않습니다.

- **목표 공유 불가:** 각자가 얼마를 모았는지 공유할 수단 없음
- **동기부여 없음:** 혼자 모으다 포기하는 경우 빈번
- **유동성 없음:** 목돈 수령 시점이 고정되어 소모임 행사 일정과 불일치
- **소모임 앱과 단절:** 소모임 플랫폼과 전혀 연결되지 않아 데이터화 불가

> **결론:** 현행 대안들은 소모임 생태계가 필요로 하는 '투명하고 안전한 공동 목적 자금 적립' 기능을 어느 것도 제공하지 못합니다. 이것이 아임인 소모임 전용 스테이지가 유일한 해결책인 이유입니다.
