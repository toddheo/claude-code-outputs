# 3.2 소모임 유형별 맞춤 활용 시나리오

**[2026-05-12]**

---

## 3.2.1 시나리오 개요

소모임의 활동 유형에 따라 아임인 스테이지 활용 방식이 달라집니다. 아래 4가지 시나리오는 소모임 플랫폼 내 주요 카테고리를 기반으로 설계했습니다.

---

## 3.2.2 시나리오 A: 등산·아웃도어 소모임 — 장비 업그레이드 펀드

**상황:** 7인 등산 소모임. 매년 한 차례 100km 울트라 트레일 원정을 가고 싶지만, 장비 교체 비용(1인당 50~80만 원)이 부담.

| 항목 | 내용 |
| ---- | ---- |
| **참여 인원** | 7명 |
| **스테이지** | 7개월 |
| **월 납입액** | 1인당 10만 원 |
| **수령액 (1인당)** | 70만 원 |
| **활동 지원금** | +6만 원 (티웨이브 지급) |
| **실제 효과** | 7명 모두 7개월 안에 장비 구매 자금 확보. 단체 원정 성사. |

> "우리 소모임은 아임인으로 모아서 드디어 지리산 종주 장비 풀세트 장만했어요. 티웨이브 지원금으로 회식도 한 번 더 했습니다."

---

## 3.2.3 시나리오 B: 여행 소모임 — 해외 여행 공동 적립

**상황:** 5인 여행 소모임. 매년 동남아 패키지 여행을 목표로 하지만, 선납금 마련이 늦어져 매번 일정이 미뤄짐.

| 항목 | 내용 |
| ---- | ---- |
| **참여 인원** | 5명 |
| **스테이지** | 5개월 |
| **월 납입액** | 1인당 20만 원 |
| **수령액 (1인당)** | 100만 원 |
| **활동 지원금** | +5만 원 (티웨이브 지급) |
| **실제 효과** | 5개월 후 5명 전원 여행 경비 확보. 항공권·숙소 동시 예약 가능. |

> "매년 '다음에 가자'가 반복됐는데, 아임인 스테이지로 모으니 이번엔 진짜로 갔습니다."

---

## 3.2.4 시나리오 C: 사진 소모임 — 고가 장비 공동 구매 펀드

**상황:** 9인 사진 소모임. 소모임 공용 망원렌즈(시가 180만 원)를 공동 구매하고 싶지만, 일시에 분담하기엔 부담.

| 항목 | 내용 |
| ---- | ---- |
| **참여 인원** | 9명 |
| **스테이지** | 9개월 |
| **월 납입액** | 1인당 20만 원 |
| **수령액 (1인당)** | 180만 원 |
| **활동 지원금** | +7만 원 (티웨이브 지급) |
| **공동 활용** | 수령한 자금 중 180만 원으로 공용 장비 구매, 나머지는 개인 장비 업그레이드 |

> "스테이지로 모아서 소모임 공용 렌즈 샀어요. 매주 모임 때마다 번갈아 쓰니 진짜 남는 장사입니다."

---

## 3.2.5 시나리오 D: 독서 소모임 — 연말 북클럽 여행 자금

**상황:** 6인 독서 소모임. 매년 연말에 1박2일 독서 캠프를 꿈꾸지만, 회비 미납·정산 갈등으로 번번이 무산.

| 항목 | 내용 |
| ---- | ---- |
| **참여 인원** | 6명 |
| **스테이지** | 7개월 (5월 시작 → 11월 수령) |
| **월 납입액** | 1인당 15만 원 |
| **수령액 (1인당)** | 90만 원 |
| **활동 지원금** | +6만 원 (티웨이브 지급) |
| **실제 효과** | 연말 북클럽 캠프 비용(숙소+식비+책 구매) 완전 해결. 정산 분쟁 없음. |

> "작년엔 회비 누가 안 냈다고 싸웠는데, 이번엔 아임인으로 모으니까 서로 눈치 볼 일이 없었어요."
