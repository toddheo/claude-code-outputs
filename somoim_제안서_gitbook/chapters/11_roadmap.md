# 4.2 단계별 추진 로드맵

**[2026-05-12]**

---

## 4.2.1 4단계 로드맵 개요

| 단계 | 기간 | 목표 |
| :--: | ---- | ---- |
| **Phase 1** | 2026년 6~8월 (3개월) | MOU 체결 및 파일럿 준비 |
| **Phase 2** | 2026년 9~12월 (4개월) | 파일럿 운영 및 성과 검증 |
| **Phase 3** | 2027년 1~6월 (6개월) | 전체 소모임으로 확대 |
| **Phase 4** | 2027년 7월~ | 상시 운영 체계 정착 |

---

## 4.2.2 Phase 1: MOU 체결 및 파일럿 준비 (2026년 6~8월)

**목표:** 파트너십 공식화 및 소모임 전용 스테이지 구축

| 항목 | 내용 | 담당 |
| ---- | ---- | :--: |
| MOU 체결 | 프렌즈큐브-티웨이브 파트너십 공식 협약 | 양사 |
| 소모임 전용 스테이지 개발 | 3~9인 소그룹 최적화, 활동 지원금 연동 | 티웨이브 |
| 소모임 앱 내 배너 셋업 | 아임인 입점 배너·배지 디자인 및 탑재 | 프렌즈큐브 |
| 파일럿 소모임 선정 | 카테고리 다양성 고려, 50~100인 규모 소모임 10개 선정 | 양사 |
| CS 프로세스 구축 | 소모임 유저 전용 문의 채널 및 응대 매뉴얼 | 티웨이브 |

---

## 4.2.3 Phase 2: 파일럿 운영 (2026년 9~12월)

**목표:** 핵심 KPI 검증 및 개선점 도출

**대상:** 파일럿 소모임 10개 × 평균 6명 = 약 60명

| KPI | 목표 |
| ---- | :--: |
| 스테이지 가입률 (대상 소모임 기준) | 70% 이상 |
| 스테이지 유지율 | 85% 이상 |
| 소모임 앱 DAU 변화 | 참여 소모임 기준 +20% 이상 |
| 이용자 만족도 | 4.0점 이상 / 5점 |
| 지원금 지급 완료율 | 100% |

**2026년 9월 론칭 목표:** 연말 행사 자금 마련을 위한 7개월 스테이지 → 2027년 4월 수령 = 가장 자연스러운 첫 캠페인.

---

## 4.2.4 Phase 3: 전체 확대 (2027년 1~6월)

**목표:** 소모임 앱 전체 유저 대상 '목돈 챌린지' 공식 기능으로 출시

| 확대 항목 | 내용 |
| --------- | ---- |
| 대상 소모임 | 소모임 앱 전체 소모임장에게 오픈 |
| 카테고리별 맞춤 캠페인 | 등산·여행·사진 등 카테고리별 시즌 캠페인 기획 |
| 소모임 앱 내 정식 탭 신설 | '목돈 챌린지' 전용 탭 또는 위젯 추가 협의 |
| 언론 공동 보도 | 양사 공동 보도자료 발표 — '국내 최초 커뮤니티 기반 공동 금융 플랫폼' |

---

## 4.2.5 Phase 4: 상시 운영 (2027년 7월~)

**목표:** 소모임 활동의 표준 루틴으로 정착

- 소모임 신규 가입자에게 '목돈 챌린지' 자동 안내
- 소모임 카테고리·시즌에 맞는 상시 캠페인 운영
- 연간 파트너십 성과 리뷰 및 수익 모델 고도화 협의
