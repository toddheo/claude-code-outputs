# 4.3 파일럿(시범 운영) 세부 계획

**[2026-05-12]**

---

## 4.3.1 파일럿 목적

파일럿은 전체 확대 전에 핵심 가설을 검증하는 단계입니다. 아래 3가지 가설을 실데이터로 검증합니다:

1. **소모임 유저는 공동 스테이지에 기꺼이 참여하는가?** (가입률)
2. **스테이지 진행 중 소모임 앱 체류 시간이 증가하는가?** (리텐션 효과)
3. **완료 후 재가입 및 주변 소모임 추천 의향이 있는가?** (바이럴 효과)

---

## 4.3.2 파일럿 소모임 선정 기준

| 기준 | 내용 |
| ---- | ---- |
| **규모** | 소모임원 5~15명 |
| **활동성** | 최근 3개월 내 오프라인 활동 1회 이상 |
| **카테고리 다양성** | 등산·여행·사진·독서·운동 등 5개 이상 카테고리 포함 |
| **디지털 친숙도** | 소모임 앱 월간 접속 3회 이상 |
| **소모임장 협조 의향** | 사전 동의 필수 |

**파일럿 규모:** 소모임 10개 × 평균 6명 = **약 60명** (2026년 9~12월, 4개월)

---

## 4.3.3 파일럿 운영 프로세스

```
[사전 준비]
1. 파일럿 소모임 10개 선정 (프렌즈큐브 협조)
2. 소모임장에게 파일럿 취지·혜택 안내
3. 소모임원 아임인 회원가입 지원 (링크 공유)

[운영 단계]
4. 소모임원 3인 이상 공동 스테이지 가입 → 자동 매칭
5. 월별 납입 진행, 티웨이브 CS팀 전담 지원
6. 프렌즈큐브에 월별 참여 현황 리포트 공유

[완료 단계]
7. 스테이지 완료 시 지원금 1인당 5~7만 원 지급
8. 이용자 만족도 설문 (NPS 포함)
9. 파일럿 결과 보고서 양사 공동 작성
```

---

## 4.3.4 KPI 측정 방법

| KPI | 측정 방법 | 목표 |
| ---- | --------- | :--: |
| **스테이지 가입률** | 소모임원 대비 아임인 가입자 수 | ≥ 70% |
| **유지율** | 스테이지 중도 이탈 없이 완료한 비율 | ≥ 85% |
| **소모임 DAU 변화** | 참여 소모임의 앱 방문 빈도 전후 비교 | ≥ +20% |
| **이용자 만족도** | 스테이지 완료 후 5점 만점 설문 | ≥ 4.0점 |
| **재참여 의향** | "다음 스테이지도 참여하겠습니까?" | ≥ 70% |
| **주변 추천 의향 (NPS)** | 순고객추천지수 | ≥ 50 |

---

## 4.3.5 파일럿 성공 시 다음 단계

파일럿 KPI 달성 시 → Phase 3 전체 확대 자동 진행.
파일럿 중 개선점 도출 시 → 개선 사항 반영 후 2차 파일럿 또는 제한적 확대 진행.
