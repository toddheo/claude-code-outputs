# 4.4 확대 방안 및 지속 운영 체계

**[2026-05-12]**

---

## 4.4.1 확대 원칙: 검증 후 단계적 확장

파일럿 성과 데이터를 기반으로, 소모임 전체 카테고리와 유저 규모로 단계적으로 확대합니다.

---

## 4.4.2 카테고리별 맞춤 확대 전략

소모임의 카테고리 특성에 따라 맞춤 캠페인을 설계합니다.

| 카테고리 | 특성 | 맞춤 접근 |
| -------- | ---- | --------- |
| **등산·아웃도어** | 장비 교체 주기 1~2년, 원정 트립 수요 | 9~11개월 장기 스테이지 추천, 시즌 전 론칭 |
| **여행** | 시즌성 강함 (봄·가을·연말), 선납금 필요 | 5~7개월 단기 스테이지, 출발 6개월 전 가입 유도 |
| **사진·크리에이터** | 고가 장비 공동 구매 수요 | 공동 구매 스테이지 특화 안내 |
| **독서·스터디** | 소규모(3~5인), 연말 행사 | 3~5인 소그룹 스테이지, 연말 행사 자금 캠페인 |
| **러닝·피트니스** | 대회 참가비·단체 물품 수요 | 대회 일정에 맞춘 단기 스테이지 |

## 4.4.3 전체 확대 목표

| 시점 | 목표 |
| :--: | ---- |
| **파일럿 완료 (2026년 12월)** | 참여 소모임 10개, 이용자 약 60명 |
| **Phase 3 완료 (2027년 6월)** | 참여 소모임 100개+, 이용자 약 600명 |
| **상시 운영 1년차 (2027년 말)** | 참여 소모임 500개+, 이용자 약 3,000명 |

## 4.4.4 지속 운영 체계

### 정기 성과 평가

| 평가 항목 | 주기 | 활용 |
| --------- | :--: | ---- |
| 참여 소모임·이용자 수 | 월별 | 프렌즈큐브 공유, 확대 속도 조정 |
| 스테이지 완료율·지원금 집행 내역 | 월별 | 예산 관리 및 지원금 단가 조정 |
| 이용자 만족도·NPS | 분기 | 서비스 개선 반영 |
| 소모임 앱 DAU 변화 | 분기 | 파트너십 효과 측정 |

### 서비스 고도화 로드맵

| 시기 | 고도화 내용 |
| :--: | ----------- |
| **도입 1년차** | 소모임 카테고리별 맞춤 스테이지 템플릿 구축 |
| **도입 2년차** | 소모임 앱 내 아임인 전용 탭 신설 협의, 공동 마케팅 확대 |
| **도입 3년차** | 소모임 전역 후(모임 해체 후)에도 이용 가능한 개인 서비스 연계 |
