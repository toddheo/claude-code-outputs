# 4.1 기관 간 협력 체계 및 역할 분담

**[2026-05-12]**

---

## 4.1.1 협력 구조 개요

소모임 × 아임인 파트너십은 **프렌즈큐브의 채널 자산**과 **티웨이브의 금융 인프라**를 결합하는 구조입니다. 프렌즈큐브는 비용 없이 유저에게 혜택을 제공하고, 티웨이브는 신규 이용자 기반을 확장합니다.

---

## 4.1.2 역할 분담 (R&R)

| 구분 | 역할 | 세부 내용 |
| :--: | ---- | --------- |
| **프렌즈큐브** | 채널 제공 | 소모임 앱 내 아임인 노출 (배너·배지·푸시 알림) |
| **프렌즈큐브** | 홍보 지원 | 소모임장 대상 '목돈 챌린지' 프로모션 안내 발송 |
| **프렌즈큐브** | 데이터 공유 | 참여 소모임 카테고리·규모 정보 공유 (동의 기반) |
| **(주)티웨이브** | 플랫폼 운영 | 아임인 스테이지 설계·운영 전담 |
| **(주)티웨이브** | 지원금 집행 | 완료 스테이지 1인당 5~7만 원 활동 지원금 직접 지급 |
| **(주)티웨이브** | CS 전담 | 소모임 유저 대상 고객 응대 100% 티웨이브 부담 |
| **(주)티웨이브** | 성과 리포트 | 월별 참여 현황·완료율·지원금 집행 내역 공유 |

---

## 4.1.3 수익 모델

| 항목 | 내용 |
| ---- | ---- |
| **프렌즈큐브 비용** | 0원 (전액 티웨이브 부담) |
| **티웨이브 수익** | 소모임 채널 통해 유입된 신규 AU의 아임인 스테이지 수수료(1.1%) 수취 |
| **향후 협의 가능** | 트래픽 규모에 따라 수수료 분배 또는 라이선스 모델 협의 |

---

## 4.1.4 소모임 전용 '목돈 챌린지' 프로모션 설계

| 구분 | 내용 |
| ---- | ---- |
| **프로모션명** | 소모임 목돈 챌린지 (가칭) |
| **참여 조건** | 소모임 앱 인증 사용자, 3인 이상 공동 가입 |
| **스테이지 옵션** | 5·7·9·11·13개월 중 선택, 월 10만~160만 원 |
| **지원금 지급 조건** | 스테이지 1회 완료 시 |
| **지원금 금액** | 1인당 5~7만 원 (규모 확인 후 확정) |
| **지급 방식** | 스테이지 완료 확인 후 계좌 직접 송금 |

## 4.1.5 운영 거버넌스

| 항목 | 내용 |
