# 3.3 안전성 및 리스크 관리 체계

**[2026-05-12]**

---

## 3.3.1 3중 안전장치 개요

아임인의 안전성은 단순한 약속이 아닌, **제도적으로 설계된 3중 구조**에 의해 보장됩니다.

```
[Layer 1] KCB 신용 확인      → 그룹 구성 단계에서 리스크 사전 차단
[Layer 2] 안심서비스 (보험)   → 미납 발생 시 즉시 대납 → 다른 멤버 영향 없음
[Layer 3] 제1금융권 에스크로  → 신한은행·케이뱅크 분리 보관 → 원금 구조적 보호
```

---

## 3.3.2 Layer 1: KCB 신용 확인

- 스테이지 참여 전 KCB(코리아크레딧뷰로) 신용 정보를 자동으로 확인합니다
- 일정 기준 이하 신용 상태인 경우 그룹 참여 제한
- 이를 통해 그룹 구성 단계에서 미납 가능성이 높은 참여자를 사전에 차단

> 소모임 유저들은 소모임 플랫폼을 통해 이미 신뢰 기반이 형성되어 있습니다. 여기에 KCB 신용 확인이 더해지면 이중 신뢰 구조가 만들어집니다.

---

## 3.3.3 Layer 2: 안심서비스 (보험 연계)

- 납입 의무가 있는 멤버가 연체 발생 시, 보험사가 해당 금액을 **자동 대납**
- 다른 멤버는 연체 여부를 인지할 필요 없이, 정해진 날짜에 정상적으로 목돈 수령
- 이 서비스는 유저 선택(Option)에 따라 가입 — 원금 무조건 보장이 아닌, 보험사의 합법적 채권 양도 구조

**결과:** 연체가 발생해도 내 돈은 100% 안전합니다.

---

## 3.3.4 Layer 3: 제1금융권 에스크로

| 파트너 | 역할 |
| ------ | ---- |
| **신한은행** | 에스크로 계좌 운영, 자금 분리 보관 |
| **케이뱅크(KBank)** | API 연동, 실시간 자금 이체·정산 |

- 모든 스테이지 자금은 티웨이브 법인 계좌가 아닌 **별도 에스크로 계좌**에 분리 보관
- 티웨이브가 경영상 문제가 생기더라도 스테이지 자금은 영향을 받지 않음

---

## 3.3.5 연체율 실적으로 증명하는 안전성

| 연도 | 연체율 | 경제 상황 |
| :--: | :----: | --------- |
| 2020 | 0.46% | COVID-19 팬데믹 |
| 2021 | 0.65% | COVID 피크 (역대 최고치) |
| 2022 | 0.52% | 고금리·고물가 정점 |
| 2023 | 0.40% | 안정화 진입 |
| 2024 | 0.26% | 지속 개선 |
| **2025** | **0.22%** | **제1금융권 수준** |

> 8년 운영 기간 중 단 한 번도 연체율 1% 초과 없음. 국내 시중은행 평균 NPL 0.4~0.8%와 비교해도 동등하거나 우수한 수준.

소모임 유저들은 이미 공동체 의식이 강합니다. '함께 가입한 소모임 멤버에게 폐를 끼치지 않겠다'는 사회적 책임감이 연체율을 더욱 낮추는 추가 방어기제로 작동합니다.
