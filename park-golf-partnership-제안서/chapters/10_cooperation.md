# 4.1 협력 체계 및 역할 분담 (R&R)

---

## 4.1.1 핵심 협업 모델

> **"아임인 스테이지 참여 = 파크골프장 혜택"의 강력한 리워드 연계 구조**

파크골프장은 고객 접점(공간·채널)을 제공하고, 아임인은 혜택의 재원(자본·금융 인프라)을 제공합니다. 각자의 강점이 완벽하게 상호 보완됩니다.

---

## 4.1.2 역할 분담표

### 제휴 파크골프장의 역할

| 역할 | 세부 내용 |
|------|---------|
| **홍보 채널 제공** | 매장 내 아임인 제휴 포스터·배너 비치 (아임인이 제작·제공) |
| **방문객 안내** | 파크골프 이용객 대상 아임인 스테이지 참여 안내 및 소개 |
| **타석 인프라 제공** | 아임인 스테이지 참여 고객에게 매장 인프라(타석) 이용 기회 제공 |
| **O2O 라운지 운영** | 직원 1명을 아임인 현장 가입 지원 담당으로 지정 (별도 교육 제공) |
| **대회 운영 협조** | 아임인 스폰서 대회 개최 시 매장 시설 제공 및 현장 운영 지원 |

### (주)티웨이브(아임인)의 역할

| 역할 | 세부 내용 |
|------|---------|
| **[개인 혜택] 무상 이용권** | 아임인 스테이지 가입 고객에게 제휴 파크골프장 무상 이용권 지급 |
| **[VVIP 혜택] 프리미엄 채** | 스테이지 N회 이상 유지 고객에게 맞춤형 프리미엄 파크골프채 증정 (150~300만 원 상당) |
| **[대회 지원] 상금·경품** | 파크골프장 주최 대회 시 참가자 스테이지 참여 조건으로 대회 상금·경품 파격 지원 |
| **[홀인원 스폰서십]** | 스테이지 참여자 홀인원 시 현금 30~50만 원 즉시 지급 |
| **[데이터 제공]** | 아임인 대안신용평가 기반 매장 회원 금융 우량 데이터 공유 (동의 고객 한정) |
| **[인센티브 지급]** | O2O 라운지 가입 유치 건수에 따른 수수료 매장에 정기 지급 |
| **[마케팅 지원]** | 아임인 공식 채널(앱·SNS·CRM)에서 제휴 파크골프장 노출 |

## 4.1.3 수익·시너지 모델

```
제휴 효과 흐름도

  파크골프 동호인
       ↓ 매장 방문
  제휴 파크골프장 (아임인 O2O 라운지)
       ↓ 스테이지 가입 안내
  아임인 스테이지 가입
       ↓
  ┌────────────────────────────────────────────┐
  │  아임인 → 고객: 무상 이용권 + 채 + 홀인원   │
  │  아임인 → 매장: 대회 상금 + 인센티브         │
  │  매장   → 아임인: 유저 유입 채널 제공         │
  │  고객   → 매장: 스테이지 기간 동안 정기 방문  │
  └────────────────────────────────────────────┘
       ↓
  파크골프장: 모객 폭증 + Lock-in + 추가 수익
  아임인:    시니어 금융 유저 대량 확보
```

## 4.1.4 운영 거버넌스

| 항목 | 내용 |
|------|------|
| **제휴 계약** | 티웨이브-파크골프장 간 B2B 파트너십 계약 체결 |
| **독점 지역 설정** | 제휴 체결 시 해당 상권 내 타 파크골프장 제휴 제한 (선점 매장 보호) |
| **성과 리포트** | 월 1회 가입자 수·유치 인센티브 정산 리포트 제공 |
| **담당 매니저** | 티웨이브 BD팀 전담 매니저 배정 (정기 방문 및 운영 지원) |
| **교육 지원** | 매장 직원 대상 아임인 서비스 설명 교육 1회 제공 |
