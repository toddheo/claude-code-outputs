# 3.2 시니어 파크골퍼 맞춤 활용 시나리오

---

> **아임인 스테이지는 파크골프 동호인의 실제 생활 패턴과 완벽히 맞아떨어집니다. 아래 4가지 시나리오로 구체적인 혜택을 확인하세요.**

---

## 시나리오 ①  "파크골프채 바꾸고 싶었던 김 씨" — 9개월 스테이지

**프로필:** 63세 남성, 퇴직 후 주 3회 파크골프 / 현재 중급자용 채 사용 중, 프리미엄 채로 업그레이드를 원하지만 200만 원 지출이 부담

| 항목 | 내용 |
|------|------|
| **스테이지** | 9개월 |
| **월 납입액** | 100만 원 |
| **수령 목돈** | 900만 원 (9명 그룹, 자신의 순번에 수령) |
| **추가 혜택** | 스테이지 유지 실적 → **맞춤형 프리미엄 파크골프채(200만 원 상당) 증정** |
| **파크골프장 이용** | 스테이지 기간 9개월 × 무상 회원권 → 9개월간 정기 방문 |

**결과:** 9개월 동안 매달 100만 원씩 저축하여 900만 원 목돈을 마련하고, 원하던 프리미엄 파크골프채까지 무상으로 받습니다. 매달 파크골프장에서 동호회 모임을 이어갑니다.

---

## 시나리오 ②  "동호회 총무 이 씨" — 동호회 그룹 참여

**프로필:** 58세 여성, 파크골프 동호회 총무 / 매주 화·목 정기 라운드 / 동호회원 9명과 함께 스테이지 참여

| 항목 | 내용 |
|------|------|
| **스테이지** | 9개월 (동호회원 9명 그룹 결성) |
| **월 납입액** | 개인별 50만 원 |
| **순번 수령액** | 450만 원 (50만 원 × 9명) |
| **그룹 혜택** | 9명 전원 무상 회원권 수령 → 9개월간 정기 모임 지속 |
| **동호회 효과** | 매달 "이번 달 누가 탔지?" 화제 → 동호회 결속력 강화 |

**결과:** 동호회 전체가 아임인 스테이지로 묶여, 9개월간 매주 같은 파크골프장에서 모입니다. 총무 이 씨는 가입 유치 수수료까지 받을 수 있습니다.

---

## 시나리오 ③  "홀인원 꿈꾸는 박 씨" — 홀인원 스폰서십 활용

**프로필:** 70세 남성, 은퇴 후 파크골프가 유일한 취미 / 홀인원 한 번이 소원이지만 "맞으면 한턱 내야 하니" 약간 부담

| 항목 | 내용 |
|------|------|
| **스테이지** | 7개월 (월 30만 원) |
| **혜택** | 아임인 스테이지 참여자 홀인원 시 **현금 30만 원 즉시 지급** |
| **효과** | "홀인원 턱" 부담 해소 → 홀인원 기록해도 오히려 이득 |
| **매장 홍보** | "우리 매장에서 홀인원하면 아임인이 현금 드립니다" |

**결과:** 박 씨는 홀인원 부담 없이 마음껏 즐기고, 파크골프장은 지역 내 유일한 "홀인원하면 현금 받는 매장"으로 입소문납니다.

---

## 시나리오 ④  "처음 가입하는 디지털 취약 최 씨" — O2O 라운지 활용

**프로필:** 74세 여성, 스마트폰 기본 사용 가능하지만 금융 앱 설치·인증이 두려움 / 동호회에서 권유받아 관심은 있음

| 항목 | 내용 |
|------|------|
| **접근 방식** | 파크골프장 직원이 라운드 후 휴식 중 15분 현장 지원 가입 |
| **지원 내용** | 앱 설치 → 본인인증 → 계좌 연동 → 스테이지 선택까지 1:1 도움 |
| **즉시 혜택** | 가입 완료 → 무상 회원권 즉시 발급 |
| **스테이지** | 5개월 (월 20만 원) → 수령 100만 원 |

**결과:** 디지털 장벽 없이 현장에서 가입 완료. 최 씨는 목돈 마련을 시작하고, 매장은 가입 유치 인센티브를 받으며 시니어 유저 허브 역할을 합니다.

---
