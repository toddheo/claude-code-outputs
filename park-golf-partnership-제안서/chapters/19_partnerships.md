## 6.3.1 제도권 금융기관 파트너십

### 신한은행

| 항목 | 내용 |
|------|------|
| **파트너십 형태** | 공식 전략적 파트너십 |
| **의미** | 국내 최대 시중은행과의 협력으로 제도권 금융 공신력 확보 |
| **활용** | 아임인 플랫폼 내 신한은행 계좌 연동, 금융 서비스 연계 |

신한은행은 국내 자산 규모 1위 은행그룹으로, 아임인과의 파트너십은 **플랫폼의 신뢰성과 안전성을 공식 보증**하는 의미를 갖습니다.

### 케이뱅크(KBank)

| 항목 | 내용 |
|------|------|
| **파트너십 형태** | 공식 전략적 파트너십 |
| **의미** | 국내 1호 인터넷전문은행과의 협력으로 디지털 금융 인프라 강화 |
| **활용** | 아임인 플랫폼 내 케이뱅크 계좌 연동, 디지털 금융 서비스 연계 |

## 6.3.2 신한퓨처스랩 10기 선정

```
신한퓨처스랩(Shinhan Future's Lab)이란?
├── 신한금융그룹이 운영하는 국내 최대 금융 스타트업 육성 프로그램
├── 매년 수백 개 기업 지원 중 극소수만 선발
├── 선정 기업: 신한금융그룹과의 직접 협업 기회 및 공신력 확보
└── 아임인(티웨이브): 10기 선정 (2017년)
```

신한퓨처스랩 선정은 국내 금융권에서 **"검증된 핀테크 기업"으로 인정받은 공식 훈장**입니다.

## 6.3.3 파트너십이 파크골프장 제휴에 주는 의미

파크골프장 대표님 입장에서 가장 중요한 질문:
> **"아임인, 믿을 수 있는 회사인가요?"**

아임인의 파트너십이 명확히 답합니다:

```
신뢰 근거 3중 검증

  ① 신한은행 공식 파트너
     → 국내 1위 시중은행이 함께하는 플랫폼

  ② 케이뱅크(KBank) 공식 파트너
     → 국내 1호 인터넷전문은행이 검증한 플랫폼

  ③ 신한퓨처스랩 10기 선정
     → 금융권 최고 권위의 프로그램이 인증한 기업

  + 8년 이상 운영, 3조 863억 원 누적 거래, 연체율 0.22%
  → 수치로 증명된 신뢰
```

동호인 어르신들께 "신한은행이랑 케이뱅크가 파트너인 앱이에요"라는 한 마디가 가장 강력한 신뢰 설득 도구가 됩니다.

## 6.3.4 언론 보도 현황

아임인은 다수의 주요 언론에 소개되었습니다.

| 매체 | 내용 |
|------|------|
| 중앙일보, 조선일보 등 | 디지털 계모임 플랫폼 혁신 사례로 소개 |
| 한국경제, 매일경제 | 핀테크 스타트업 성장 사례 |
| 신한금융그룹 공식 채널 | 신한퓨처스랩 선정 기업 소개 |

지속적인 언론 노출은 아임인의 **사회적 인지도와 브랜드 신뢰성**을 뒷받침합니다.
