# 4.2 단계별 추진 로드맵

---

## 전체 로드맵 개요

```
Phase 1          Phase 2           Phase 3          Phase 4
준비·계약         파일럿 운영         성과 검증·확대    정착·고도화
(1~2개월)        (3~6개월)          (7~12개월)        (12개월~)

  계약 체결   →   O2O 라운지   →   KPI 달성 확인  →  프랜차이즈
  환경 세팅        첫 대회 개최      확대 협의         본사 제안
  직원 교육        홀인원 스폰서      독점 지역 확대     모델 정착
```

---

## Phase 1. 준비·계약 (D-Day ~ +2개월)

### 목표: 제휴 환경 완성 및 첫 고객 가입 준비

| 주차 | 티웨이브 | 파크골프장 |
|------|---------|----------|
| **1~2주** | 계약서 초안 협의·검토 | 내부 의사결정 및 계약 승인 |
| **3~4주** | 포스터·배너·리플릿 제작·납품 | 매장 내 아임인 홍보물 설치 공간 확보 |
| **5~6주** | 직원 교육 (아임인 서비스 설명, O2O 가입 프로세스) | 담당 직원 1~2명 지정 및 교육 참여 |
| **7~8주** | 독점 지역 설정 완료, 첫 대회 일정 협의 | 첫 대회(파일럿) 참가자 모집 시작 |

### 마일스톤
- [ ] 파트너십 계약 체결
- [ ] 매장 내 홍보물 설치 완료
- [ ] 직원 교육 이수 완료
- [ ] 파일럿 대회 일정 확정

---

## Phase 2. 파일럿 운영 (3~6개월)

### 목표: 첫 대회 개최 + O2O 라운지 가동 + 초기 유저 확보

| 활동 | 내용 |
|------|------|
| **O2O 라운지 정식 가동** | 매장 직원이 파크골프 라운드 후 휴식 중 방문객에게 현장 가입 지원 시작 |
| **파일럿 대회 개최** | 아임인 스테이지 참여 조건 대회 (총상금 500만~1,000만 원 규모) |
| **홀인원 스폰서십 공지** | 매장 내 "아임인 스테이지 참여자 홀인원 시 현금 지급" 공지 게시 |
| **가입자 모니터링** | 월별 가입자 수·이탈률·재방문율 데이터 수집 |

### 파일럿 KPI (3개월 기준)
| KPI | 목표치 |
|-----|--------|
| 누적 스테이지 가입자 | **30명 이상** |
| O2O 라운지 가입 건수 | **월 10건 이상** |
| 매장 재방문율 변화 | **기준치 대비 +20% 이상** |
| 대회 참가자 중 스테이지 전환율 | **40% 이상** |

---

## Phase 3. 성과 검증·확대 (7~12개월)

### 목표: 파일럿 KPI 달성 확인 후 제휴 심화·규모 확대

| 활동 | 내용 |
|------|------|
| **대형 대회 확대** | 총상금 2,000만~3,000만 원 규모 대회로 업그레이드 |
| **VVIP 혜택 발동** | 스테이지 유지 실적 우수 고객에게 맞춤형 프리미엄 채 증정 시작 |
| **데이터 공유** | 아임인 대안신용평가 기반 VIP 회원 데이터 매장 제공 |
| **추가 매장 확대** | 인근 지역 추가 파크골프장과 제휴 논의 (독점 지역 확장) |

---

## Phase 4. 정착·고도화 (12개월~)

### 목표: 프랜차이즈 본사와 전국 단위 제휴 확대

- 단일 매장 성공 사례를 바탕으로 **프랜차이즈 본사(레저로·GTR·피닉스 등)에 전국 제휴 제안**
- 아임인 × 파크골프 브랜드 공동 마케팅 캠페인 기획
- 파크골프 대회 시리즈 "아임인 파크골프 챔피언십" 론칭 검토
