﻿# 3. 상권 분석

[구로구 킹콩식자재마트 입점 맥락]

3.1 구로구 입지 특성

항목 내용
주거 특성 중산층 실거주 밀집 (아파트 단지 다수)
연령 구성 3040대 가족 + 고령층 혼재 → 약국 수요 안정적
유사 상권 금천구 인접 (메가팩토리약국 금천점 입점)
마트 성격 장보기 목적 방문객 중심 → 고빈도 재방문
경쟁 현황 마트 반경 500m 내 소규모 약국 → 가격 경쟁 우위 확보 가능

3.2 일일 방문객 추정 모델

ℹ️ 추정 근거: 대형 식자재마트 업계 일반적 트래픽 모델 적용. 킹콩마트 자체 현장 조사로 검증 필요.

구분 추정 수치 산출 근거
마트 일일 방문객 500~800명 중형 식자재마트 (구로구 주거 상권)
약국 전환율 15~20% 마트 방문객 중 약국 이용 비율
일일 약국 방문객 75~160명 500×15% ~ 800×20%
목표 일일 방문객 100명 정상 시나리오 기준
객단가 25,000원 강남역(28,571원) 대비 구로 할인 적용

참조: 강남역 약국 실데이터 (2024 카드매출)

항목 수치

3.3 마트 입점 vs 일반 독립 약국 비교

[일반 독립 약국]          [킹콩마트 입점 약국]
─────────────────────────────────────────────
마케팅 비용 필요      →   마트 방문객 자연 유입 (Zero CAC)
처방전 의존도 高      →   OTC + 건기식 비중 자유롭게 확대
고객당 체류시간 단    →   마트 동선으로 자연스러운 체류 증가
임대료 높은 1층 입지  →   마트 내부 계약으로 협상력 우위
재방문 동기 약함      →   장보기 + 약국 = 복합 방문 습관 형성

3.4 상권 리스크 요인

❌ Stop 조건 체크: 아래 항목은 투자 전 반드시 현장 실사로 확인해야 합니다.

리스크 내용 대응
마트 방문객 부족 실제 일일 방문객 300명 미만일 경우 BEP 달성 불가 현장 직접 카운팅 조사 필수
경쟁 약국 인접 반경 300m 내 대형 약국 기존재 사전 경쟁 실사
마트 집객력 약화 킹콩마트 영업 부진 시 약국 트래픽 동반 하락 임대차계약 해지 조건 명확화
의원 연계 부재 인근 내과·이비인후과 없을 경우 처방전 비중 극소 OTC+건기식 중심으로 전략 선회
