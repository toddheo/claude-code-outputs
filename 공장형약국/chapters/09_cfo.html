9. 수진 CFO 보상 모델 및 수익 배분

9.1 수진 CFO 역할 정의

수진은 단순 직원이 아닌 사업의 공동 운영자(Co-Operator) 로 설계. 약사(장모님)의 의료·조제 전문성과 수진의 재무·경영 역량을 결합한 구조.

역할 영역 세부 내용
재무·회계 총괄 POS·ERP 연동 실시간 매출 관리, 월간 재무제표 작성, 세무신고 지원
법무·계약 관리 동업계약·임대차계약·도매상 계약 관리, 법무사·변호사 커뮤니케이션
인사·노무 근무약사·관리인원 채용·근태 관리, 급여 정산, 4대보험 처리
마케팅·브랜딩 SNS 운영, 오픈 이벤트 기획, 비급여 상담 예약 시스템 운영
투자자 보고 허진영(사위)에게 월 1회 재무 보고, 의사결정 지원

9.2 추천 보상 구조: 기본급 + 영업이익 연동 인센티브

수진 CFO 확정 보상 구조
────────────────────────────────────────
기본 경영자문료  : 250만원/월 (고정, 영업이익 무관)
성과 인센티브   : 영업이익의 15% (분기별 정산)
────────────────────────────────────────

✅ 기본급 250만원: 경력자 CFO·재무담당자 시장 급여(300~500만원) 대비 합리적 수준. 성과급은 순이익이 아닌 영업이익 기준으로 설정하여 세금·감가상각 조작 없이 투명한 정산 가능.

모델 구조 월 예상 수익 (안정기) 수진 입장 장모님 입장
A. 고정 자문료만 월 300~500만원 고정 300~500만원 안정적 초기 비용 부담
B. 순이익 % 연동 순이익의 25~35% 불안정 초기 리스크 비용 예측 어려움
C. 혼합 모델 ✅ 기본 250만원 + 영업이익 15% 1,052만원 균형적 성과 연동, 공정
D. 대여금 이자 이자 수익만 38~58만원 너무 낮음 부담 없음

허진영(사위) 수익 구조

항목 내용
직접 배당 없음 (명의 미포함)
대여금 원금 사업 종료·Exit 시 1억~1.5억원 우선 회수 (공정증서 보호)
간접 혜택 1억 무이자 → 기회비용 연 460만원 절감 (기준이자율 4.6% 기준)

✅ 핵심 구조: 허진영은 원금 보호 (공정증서 + 질권) + 거부권 행사 + Exit 시 프리미엄 수취. 수진은 고정 생활비 보장(250만원) + 성과 연동 업사이드(최대 월 2,500만원+).

9.5 세금 최적화 구조

주체 소득 형태 세율 비고
장모님 사업소득 (약국 순이익) 종합소득세 15~35% (과세표준 구간별) 급여 500만원 → 필요경비 처리
수진 용역비/경영자문료 프리랜서 3.3% 원천징수 + 연말정산 사업자등록 시 부가세 환급 가능
허진영 이자소득 없음 (무이자) 해당 없음 증여세 과세 기준 이하 (연 460만원)

⚠️ 수진 사업자등록 권장: 경영자문 용역으로 세금계산서 발행 시 약국 측에서 VAT 매입공제 가능. 수진은 종합소득세 신고 시 필요경비(교통비·통신비·교육비) 공제 가능.

약국 전산 접근 권한 설정 POS·ERP·회계 시스템 수진 접근 권한 설정 🟠 긴급 개업 시
