# 2.1 아임인 SNBL — 3중 딜레마를 동시에 해결하는 구조

## 2.1.1 SNBL(Save Now, Buy Later)이란

> **SNBL은 '선(先)저축, 후(後)구매'의 원리로 작동하는 공동체 금융 모델입니다. BNPL(Buy Now, Pay Later)의 정반대 개념으로, 충동 소비 대신 계획 저축을 유도합니다.**

아임인의 SNBL 구조:

```
[참여자 9명 예시 — 9개월 스테이지, 월 50만 원 납입]

  월 1회차: 참여자 ①이 450만 원(9명 × 50만원) 수령
  월 2회차: 참여자 ②가 450만 원 수령
  ...
  월 9회차: 참여자 ⑨가 450만 원 수령
  
  결과:
  ┌──────────────────────────────────────────────────┐
  │  참여자 전원: 9개월간 월 50만 원 납입 (총 450만 원)  │
  │  각자 1회: 450만 원 목돈 즉시 수령                  │
  │  은행 이자 부담: ZERO                               │
  │  I-CSS AI: 계주 도주 리스크 원천 차단               │
  └──────────────────────────────────────────────────┘
```

## 2.1.2 딜레마별 해결 메커니즘

### 해결책 ① Zero-Cost Funding (수신 역설 해결)

```
기존 디지털뱅크                    아임인 SNBL 파트너십
    ↓                                      ↓
고금리 파킹통장 제공               아임인 스테이지 납입금을
(역마진 발생)                    파트너 은행 계좌에 적립
                                 (이자 부담 ZERO)
    ↓                                      ↓
월 이자 비용 발생                  월 이자 비용 = 0원
체리피커 이탈 시 손실              중도 이탈 = 동료에게 피해
                                 → 자발적 유지
```

파트너 은행은 아임인 유저들의 매월 정기적인 납입금을 **요구불예금(Low-cost Deposit)** 형태로 확보합니다. 단 1원의 마케팅 비용이나 이자 없이.

### 해결책 ② Community Lock-in (체리피커 차단)

아임인의 핵심 차별점은 **피어 프레셔(Peer Pressure) 시스템**입니다.

| 구분 | 일반 금융 상품 | 아임인 스테이지 |
|------|-------------|--------------|
| 중도 해지 | 가능 (위약금 소액) | 사실상 불가 (동료 전체 피해) |
| 이탈 동기 | 더 좋은 혜택 출현 시 이탈 | 인간관계 훼손 우려 → 유지 |
| 연체율 | 5~15% (씬파일러) | **0.22%** (8년 실증) |

### 해결책 ③ Alpha Credit 파이프라인 (여신 병목 해결)

```
아임인 스테이지 완료자 데이터

  ✅ 5~13개월간 매월 정시 납입 기록
  ✅ 약속된 금액을 지키는 재무 규율 검증
  ✅ 동료와의 약속을 지키는 신뢰도 검증
  
  → Alpha Credit Score 생성
  → 기존 신용평가로 거절됐을 씬파일러 중
     '진짜 우량 고객(Hidden Prime)' 선별
  → 파트너 은행의 대출·투자 상품으로 즉시 송객
```

## 2.1.3 B2B2C 파트너십 구조

```
[아임인 B2B2C 플로우]

  아임인 유저  →  스테이지 납입  →  파트너 은행 계좌
      ↓                               ↓
  목돈 수령 후              Alpha Credit 스코어링
  재테크 욕구 최고조                  ↓
      └──────────→  파트너 은행 금융 상품 연결  ←──┘
                   (예금/대출/투자/보험)
```

**파트너 은행 입장에서 아임인은:**
1. **저비용 예금 파이프라인** — 역마진 없는 요구불예금 유동성
2. **검증된 고객 공급자** — 체리피커 아닌, 신뢰도 검증된 진성 유저
3. **대출 리스크 제로 파이프라인** — Alpha Credit으로 선별된 Hidden Prime
