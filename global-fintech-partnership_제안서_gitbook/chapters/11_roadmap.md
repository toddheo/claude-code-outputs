# 4.1 실행 로드맵

---

## Phase 1 — 파일럿 설계 (D-Day ~ D+3개월)

| 단계 | 내용 |
|------|------|
| **계약 체결** | MOU 또는 LOI 서명, 기술 연동 범위 합의 |
| **API 연동** | 파트너 은행 계좌 연결, 에스크로 계좌 설정 |
| **규제 검토** | 현지 금융 당국 사전 협의 (HKMA/FSC/MAS/PBOC 등) |
| **파일럿 대상** | 파트너 은행 기존 고객 1,000~5,000명 선정 |
| **현지화** | 언어(중문/영문), 문화(표회 맥락 마케팅) 적용 |

## Phase 2 — 파일럿 운영 (D+3개월 ~ D+9개월)

```
파일럿 KPI

  ✅ 스테이지 가입자 1,000명 이상
  ✅ 완료율(이탈 없이 스테이지 완주) 85% 이상
  ✅ Alpha Credit 기반 대출 전환율 15% 이상
  ✅ 파트너 은행 신규 금융 상품 가입률 20% 이상
  ✅ NPS(순추천지수) 60점 이상
```
