# 1.2 아시아 디지털뱅크의 3중 딜레마

---

> **아시아 디지털뱅크들은 구조적으로 동일한 3가지 딜레마에 갇혀 있습니다. 개별 처방으로는 탈출이 불가능합니다.**

---

## 딜레마 ① "수신(예금)의 역설" — 이자를 퍼주지 않으면 돈이 모이지 않는다

### 문제의 본질

```
일반 디지털뱅크의 고객 유치 공식

  신규 가입 보너스 (SGD 50~200 또는 대만 NT$ 500~2,000)
  + 고금리 파킹통장 (기준금리 + 1~2%p)
  + 캐시백·포인트 제공
  
  → 월 고객 유치 비용 (CAC): 1인당 평균 USD 50~150
  → 프로모션 종료 후 이탈률: 60~80%
  → 실질 비용 회수 기간: 18개월 이상 (흑자 전환 불가)
```

| 항목 | 내용 |
|------|------|
| **근본 문제** | 고객이 "혜택" 때문에 온다 → 혜택이 끝나면 떠난다 |
| **결과** | 고금리 유지 시 역마진, 금리 인하 시 즉각 이탈 |
| **현재 최선책** | 더 큰 혜택 제공 → 더 깊은 적자 수렁 |

**핵심 문제:** 디지털뱅크가 아무리 금리를 올려도, 고객은 금리를 위해 왔을 뿐 은행을 위해 남지 않습니다.

---

## 딜레마 ② "가짜 성장의 함정" — 체리피커에게 CAC를 강탈당한다

### 문제의 본질

```
체리피커(Cherry Picker) 생애주기

  가입 → [사이닝 보너스 수령] → [고금리 한도 채움] → [프로모션 종료] → 이탈
  
  소요 기간: 평균 3~6개월
  은행 순손실: 1인당 USD 80~200 (CAC + 역마진)
  재가입 패턴: 신규 혜택 출시 시 재가입 → 반복
```

**핵심 문제:** 규모가 커질수록 체리피커도 늘어납니다. MAU가 증가해도 실질 수익성은 악화됩니다.

---

## 딜레마 ③ "여신(대출)의 병목" — 씬파일러가 너무 많아 돈을 빌려줄 수 없다

### 문제의 본질

디지털뱅크의 핵심 수익원은 예대마진(Net Interest Margin)입니다. 그러나:

| 구분 | 내용 |
|------|------|
| **씬파일러 비중** | 디지털뱅크 고객의 30~50%는 기존 신용평가 데이터 부족 |
| **NPL 리스크** | 씬파일러 대출 시 NPL 2~5% → 은행 건전성 악화 |
| **대출 거절률** | 높은 거절률 → 고객 이탈 → MAU 감소 악순환 |
| **결과** | 예금은 쌓이지만 대출 파이프라인 막힘 → NIM 압박 |

**핵심 문제:** 예금은 유치했지만, 안전하게 빌려줄 사람을 찾지 못합니다.

---

## 3중 딜레마 요약

```
┌─────────────────────────────────────────────────────────────────────┐
│  딜레마        │  증상                  │  결과                       │
├─────────────────────────────────────────────────────────────────────┤
│  ① 수신 역설  │  역마진 파킹통장        │  만성 적자                  │
│  ② 가짜 성장  │  체리피커 이탈          │  CAC 낭비·LTV 붕괴          │
│  ③ 여신 병목  │  씬파일러 대출 불가     │  NIM 압박·수익화 한계       │
└─────────────────────────────────────────────────────────────────────┘

공통 근본 원인: 금융 행동 데이터 부재 → 고객의 '신뢰도'를 증명할 수단 없음

아임인 SNBL은 이 세 가지를 단 하나의 구조로 동시에 해결합니다.
```
