## 2. 타겟 유저

- **최종 이용자:** 캠핏 기존 유저 및 잠재 캠퍼 — 가족·친구·직장동료 소그룹 단위 캠핑 목적 저축 수요자
- **규모:** MAU 100만+, 연간 이용자 600만 명, 누적 예약 400만 건
- **특성:**
  - 캠핑 인구 대비 93% 도달률 — 사실상 국내 캠퍼 전체를 커버하는 플랫폼
  - 목적 소비 성향 강함 (캠핑 = 계획적 지출: 사이트비·장비·식품 세트)
  - 소그룹(3~5인) 캠핑이 주류 → 공동 자금 마련 니즈 내재

## 3. 핵심 페인포인트 (캠핏이 겪는 문제)

1. **비수기 리텐션 공백** — 캠핑은 계절성이 극심한 서비스. 봄·가을 성수기 외 겨울·장마철에 MAU가 급락하는 구조적 한계. 비수기에도 앱을 열게 할 동기가 없음.

2. **목적성 자금 마련 솔루션 부재** — 캠핏은 예약·결제를 다루지만, 캠핑 비용을 '어떻게 모을지'에 대한 솔루션은 없음. 유저는 캠핑 자금을 단순 개인 적금이나 일시 지출로 처리. 공동 저축→공동 소비의 연결 고리가 끊겨 있음.

3. **신규 유저 유입 비용 증가** — 국내 캠핑 인구 약 546만 명 중 93%가 이미 캠핏 유저. 기존 캠핑 인구 포화로 신규 유저 확보를 위해서는 비캠퍼 전환이 필요하지만 CAC가 증가하는 구조.

## 4. 현행 대안의 한계

| 현행 대안 | 한계 |
|----------|------|
| 개인 은행 적금 | 공동 목표 없음 → 중도 해지율 높음, 캠핑 타이밍과 수령 시기 불일치 |
| 그룹 내 개인 계좌 관리 | 분쟁 리스크, 누가 얼마나 모았는지 투명하지 않음 |
| 쿠폰/포인트 적립형 리워드 | 소액 할인에 그침 — 30만 원 이상의 실질적 캠핑 자금 마련 불가 |
| 현행 캠핏 혜택 구조 없음 | 비수기에 앱을 열어야 할 이유 자체가 없음 |

## 5. 협업 모델 (R&R)

### 1단계: 바터(Barter) 트래픽 교환 제휴 (즉시 실행 가능, 시스템 연동 불필요)

| 역할 | 내용 |
|------|------|
| **캠핏 역할** | 자사 플랫폼(앱/웹)에 아임인 제휴 배너 노출 |
| **티웨이브 역할** | 아임인 플랫폼에 캠핏 제휴 배너 노출 |
| **혜택 구조** | 캠핏→아임인 유입 전환 시: 아임인 전용 쿠폰 4~5만 원 지급 (티웨이브 부담) / 아임인→캠핏 유입 시: 캠핏 할인 쿠폰 (캠핏 제공) |
| **수익 모델** | 상호 무상 제공 (비용 0원 구조, CPA 전환 기반 혜택 지급) |
| **의사결정 속도** | 결정 즉시 1~2주 내 실행 가능 |

### 2단계: 화이트 라벨링 — 캠핑 목적 공동 저축 모듈 이식

| 역할 | 내용 |
|------|------|
| **캠핏 역할** | 플랫폼(앱/웹) 내 '캠핑 자금 모으기' 모듈 지면 제공 |
| **티웨이브 역할** | 아임인 ROSCA 금융 시스템 화이트 라벨링 형태로 이식, 운영·CS 전담 |
| **브랜딩** | 캠핏 자체 브랜드 서비스로 운영 가능 ('Powered by imin' 표기 여부 협의) |
| **핵심 혜택** | 공동 저축 목표 달성 그룹에게 아임인 캠핑 지원금 인당 5~7만 원 지급 (5인 기준 약 25~35만 원, 티웨이브 전액 부담) |
| **수익 모델** | 캠핏 비용 0원 / 티웨이브: 신규 AU 확보·리텐션 효과 |

## 6. 기대 효과 (캠핏 입장)

- **정량적:**
  - 비수기(11~3월) DAU 유지율 개선 — 저축 기간(5~13개월) 내 주기적 앱 재방문 동기 부여
  - 신규 유저 확보 비용 절감 — 아임인 유저(60만 7,169명) 中 잠재 캠퍼 풀 유입
  - 유저 1인당 캠핑 소비 증가 — 목적성 자금 마련 완료 시 평균 객단가 상승 기대

  - '캠핑 자금까지 해결해주는 플랫폼' 브랜드 포지셔닝 강화
  - 캠핏 비용 0원으로 대규모 유저 혜택 제공 → PR·바이럴 효과

## 9. 추가 메모

**통화 내용 요약 (2026.05.12 백재학 팀장)**
- 현재 캠핏 내부적으로 4~5개 프로젝트에 집중 중이라 의사결정이 빠르지 않을 수 있음
- 백재학 팀장이 출장 중이라 미팅 일정 잡기 어려운 상황 → 이메일로 제안 내용 정리 후 마케팅 담당자 포함 공유 요청
- **바터(1차) 먼저 제안해서 빠르게 실행 후, 화이트라벨(2차)로 단계적 확대가 현실적인 접근법**
- 백재학 팀장: "내용 정리해서 이메일 주시면 내부 공유 후 의견 드리겠다"

**타임라인 고려사항**
- 2026 여름 성수기(7~8월) 전 바터 제휴 론칭이 가장 효과적
- 화이트라벨링은 시스템 연동 필요 → 최소 2~3개월 준비 기간 필요

**경쟁 상황**
- 캠핏이 캠핑 시장 93% 도달 — 사실상 독점 플랫폼으로 협상력 있으나, 그만큼 다양한 제휴 요청을 받는 위치
- 아임인 유저(목돈 마련 니즈)와 캠핏 유저(목적 소비 니즈)의 교집합이 핵심 논거

**주의사항**
- 화이트라벨링 제안 시 시스템 연동 복잡도에 대한 우려 있음 → 1차 바터 제휴로 신뢰 쌓은 뒤 자연스럽게 2차 제안
- '아임인' 브랜드 노출 여부 ('Powered by imin') 협의 여지 열어두기
