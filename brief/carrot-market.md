## 2. 타겟 유저

- **최종 이용자:** 당근마켓 기존 유저 — 동네 이웃 소그룹 단위로 모임 자금·여행 자금·공동 구매 자금을 마련하고자 하는 2030~4050 지역 생활자
- **규모:** 국내 MAU 2,000만+, 글로벌 3천만 사용자 (Series D, 누적 투자 2,259억 원, K-유니콘)
- **특성:**
  - 당근페이 주 사용자: 3040 남성 (재구매율 51.3%, 연평균 구매 12.8회)
  - 하이퍼로컬 플랫폼: 지역 인증 필수 → 동네 기반 신뢰 관계 형성이 DNA
  - 동네생활(모임·커뮤니티) 서비스 보유 → 소그룹 공동 목적 소비 니즈 내재
  - 당근알바 유저 (단기 수입자): 불규칙 소득 → 목돈 마련 수요 강함

## 3. 핵심 페인포인트 (당근마켓이 겪는 문제)

1. **당근페이 금융 생태계 확장 필요** — 2024년 당근페이 출시 후 매출 37억(전체 매출 1,892억 중 2%). 광고 의존 수익 구조(광고 1,888억)에서 금융 서비스로 수익 다각화가 절실하나, 금융 상품 라인업이 빈약함.

2. **동네생활 모임의 자금 관리 솔루션 부재** — 당근 동네생활에서 이웃과 모임을 구성해도 '함께 자금을 모으는' 기능은 없음. 회비 걷기·여행 모금·공동 구매 등은 여전히 카카오페이 송금·개인 통장으로 처리 → 당근 생태계 이탈.

3. **비거래 시간대 앱 체류 동기 부족** — 중고거래는 필요 시 방문 서비스 → 비거래 기간 중 당근 앱을 열 이유가 없음. 매주 납입·잔액 확인 같은 금융 행동이 정기적 앱 방문 동기를 만들 수 있음.

| 당근페이 현재 기능 | 결제·송금 중심. 목적성 공동 저축 기능 없음 |

## 5. 협업 모델 (R&R)

### 1단계: 바터(Barter) 트래픽 교환 제휴

| 역할 | 내용 |
|------|------|
| **당근마켓 역할** | 당근 앱 내(동네생활·당근페이 탭) 아임인 제휴 배너 노출 |
| **티웨이브 역할** | 아임인 플랫폼에 당근마켓 제휴 배너 노출 |
| **혜택 구조** | 당근→아임인 전환 시: 아임인 전용 스테이지 쿠폰 5만 원 (티웨이브 부담) / 아임인→당근 전환 시: 당근머니 쿠폰 (당근 제공) |
| **수익 모델** | 상호 무상 제공, CPA 전환 기반 |
| **의사결정 속도** | 결정 즉시 1~2주 내 실행 가능 |

### 2단계: 당근페이 × 아임인 수령금 연동

| 역할 | 내용 |
|------|------|
| **당근마켓 역할** | 당근페이 SDK·API 연동 제공, 수령금 당근머니 전환 기능 |
| **티웨이브 역할** | 아임인 ROSCA 시스템 운영, 스테이지 만기 수령금을 당근머니로 전환 옵션 제공 |
| **핵심 혜택** | 수령금을 당근머니로 전환 시 추가 보너스 (예: 수령액의 3~5% 당근머니 추가) — 당근 플랫폼 내 소비 완결 |
| **수익 모델** | 수수료 분배 협의 / 당근: GMV 증가 + 당근페이 거래량 증가 |

### 3단계: 동네생활 모임 × 아임인 화이트라벨링 (장기)

| 역할 | 내용 |
|------|------|
| **당근마켓 역할** | 동네생활 탭 내 '모임 자금 모으기' 지면 제공 |
| **티웨이브 역할** | 아임인 ROSCA 금융 시스템 화이트라벨링 이식, 운영·CS 전담 |
| **브랜딩** | '당근 모임통장' 등 당근 자체 브랜드로 운영 가능 |
| **수익 모델** | 당근 비용 0원 / 티웨이브: 신규 AU 확보·수수료 수취 |

## 6. 기대 효과 (당근마켓 입장)

- **정량적:**
  - 당근페이 금융 수익 다변화 — 현재 37억에서 금융 상품 라인업 추가로 수익 확장
  - 동네생활 MAU 증가 — 공동 저축 기간(5~13개월) 동안 정기적 앱 재방문 동기 부여
  - 당근머니 거래량 증가 — 아임인 수령금의 당근머니 전환으로 GMV 부스트
  - 아임인 유저 60만 7,169명의 당근마켓 유입 (특히 목돈 마련 니즈 보유층)
- **정성적:**
  - '하이퍼로컬 금융 플랫폼' 포지셔닝 강화 — 거래를 넘어 이웃과 함께 저축하는 커뮤니티
  - 당근페이 론칭 이후 금융 생태계 완성도 제고
  - 동네생활 서비스 차별화 — 타 플랫폼에 없는 '이웃 공동 저축' 기능

## 7. 슬로건 후보

- **"우리 동네 이웃이랑, 함께 모아요"** — 하이퍼로컬 DNA + 공동 저축 강조
- **"당근에서 모아서, 당근에서 쓴다"** — 당근 생태계 내 금융 완결 루프
- **"동네 모임, 이제 자금부터 함께"** — 동네생활 모임 × 공동 저축 연결

## 9. 추가 메모

**핵심 논거: 당근의 하이퍼로컬 DNA × 아임인의 소셜 금융**
- 당근마켓의 본질은 '이웃 간 신뢰 기반 거래'. 아임인의 ROSCA도 '소그룹 신뢰 기반 공동 저축'. 두 서비스의 DNA가 동일.
- '동네 이웃끼리 함께 모으는 계' — 디지털 시대 동네 계모임의 부활이라는 스토리 가능.

**타임라인 고려사항**
- 1단계 바터: 결정 즉시 2주 내 집행 가능 (시스템 연동 불필요)
- 2단계 당근페이 연동: API 연동 필요 → 1~2개월 준비
- 3단계 화이트라벨링: 3~6개월 개발·법무 검토 필요

**주의사항**
- 당근마켓은 내부 프로덕트 우선순위가 높고 대규모 조직 → 의사결정 사이클 길 수 있음
- 당근페이 연동 제안은 핀테크 규제 검토 필요 (티웨이브 법무팀 사전 확인 권고)
- '금융감독원', '규제 샌드박스' 표현 금지
- 아임인 서비스는 만 19세 이상 성인만 이용 가능 — 미성년자 제외 명시 필요
