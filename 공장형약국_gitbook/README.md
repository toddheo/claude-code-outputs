## 실데이터 근거 (6가지)

> ✅ **전국 약국 평균 연매출 12억 3,620만원** — 카드매출 실데이터, 데일리팜

> ✅ **공장형 약국 마진율 43%** — OTC+건기식+비급여 혼합 모델 기준

> ✅ **창고형·공장형 약국 전국 50개 이상 허가** (2025.06~2026.04, 약 10개월 내)

> ✅ **메가팩토리약국 금천점 870평** (홈플러스 내), 성남 메가팩토리 500평 일 300명+ 방문
