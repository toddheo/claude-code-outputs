## 5.1 마케팅 전략 개요

공장형 약국의 마케팅 목표는 단순하다: **마트 방문객을 약국으로 전환**하고, **마트 독립적인 비급여 집객**을 만드는 것.

```
마케팅 2축 전략
────────────────────────────────────────────────────────────
축 1: 인바운드 전환 극대화 (마트 트래픽 활용)
       → 약국 파사드·POP 마케팅으로 마트 동선에서 자연 유입
       → 마트 공동 프로모션 이벤트 (쿠폰 교환, 공동 할인행사)
       → 목표: 마트 방문객 전환율 15% → 25% 끌어올리기

축 2: 아웃바운드 독립 집객 (마트 의존도 분산)
       → SNS·네이버 검색광고로 비급여 예약 상담 고객 직접 유치
       → 제휴 병원 처방 → 약국 연계 시스템 구축
       → 목표: 일 방문객 중 독립 집객 비중 30% 이상
────────────────────────────────────────────────────────────
```

## 5.2 오픈 마케팅 전략 (M-0 ~ M+3)

### 오픈 이벤트 설계

| 이벤트명 | 내용 | 기대 효과 | 예산 |
| -------- | ---- | --------- | ---- |
| **균일가 이벤트** | 인기 OTC·비타민 5,000원 균일가 (3일) | 초기 유입 극대화, SNS 입소문 | 50~100만원 |
| **건기식 시음 부스** | 브랜드사 협찬 건기식 시음·샘플 증정 | 건기식 인지도 + 0원 비용 | 0원 (협찬) |
| **비급여 무료 상담** | 비만·탈모 1:1 무료 상담 (예약제, 10명/일) | 비급여 잠재 고객 발굴 | 0원 |
| **반려동물 존 기념** | 반려동물 간식 증정 (방문객 한정 100명) | 반려인 고객 유입 | 20~30만원 |
| **마트 공동 쿠폰** | 킹콩마트 구매 영수증 → 약국 5% 할인 | 마트↔약국 트래픽 교환 | 협찬 협의 |
| **SNS 바이럴 이벤트** | 약국 방문 인증샷 → 경품 추첨 | SNS 확산, 인지도 확보 | 30~50만원 |

> ✅ **벤치마크:** 성남 메가팩토리약국 오픈 3개월 만에 일 방문객 200명 달성. 핵심: 오픈 이벤트 SNS 바이럴 + 병원 제휴 사전 체결.

## 5.4 비급여 특화 마케팅 (핵심 차별화)

### 비만·탈모 클리닉 제휴 마케팅

```
비급여 양방향 리퍼럴 플로우
─────────────────────────────────────────────────────────
[제휴 병원]
 ├ 진료 후 약 처방 → 환자에게 약국 QR 코드·쿠폰 제공
 └ 병원 대기실에 약국 안내 팸플릿 배치

[약국]
 ├ 비만·탈모 상담 고객 → 제휴 병원 연결 (양방향 리퍼럴)
 └ 비급여 상담 예약 시스템 운영 (네이버 예약·카카오 예약)
─────────────────────────────────────────────────────────
```

### 비급여 상담 예약제 설계

| 항목 | 내용 |
| ---- | ---- |
| 예약 채널 | 네이버 예약 (무료) + 카카오 채널 |
| 상담 시간 | 평일 오전 10~12시, 오후 2~4시 (1일 최대 10명) |
| 상담 품목 | GLP-1 비만약·식욕억제제·탈모약·피부 비급여 |
| CRM 활용 | 상담 이력 ERP 등록 → 3개월 후 재구매 알림 문자 발송 |
| 전환율 목표 | 예약 상담 70% → 구매 전환 (객단가 10~20만원) |

> ⚠️ **법적 주의:** 제휴 병원 커미션 지급은 의료법 리베이트 규정 저촉 가능. Chapter 06 법적 고려사항 참조.
