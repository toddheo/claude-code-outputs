# 9. CFO 보상 모델 및 수익 배분

***

## 9.1 수진(CFO) 보상 옵션 비교

| 모델 | 구조 | 월 예상 수익 | 장점 | 리스크 |
| ---- | ---- | ----------- | ---- | ------ |
| **A. 고정 경영자문료** | 순이익 무관, 월 고정 | 300~500만원 | 안정성 | 손실기에도 비용 발생 |
| **B. 순이익 % 연동** | 순이익의 25~35% | 흑자 시 300~500만원 | 동기부여 | 초기 수익 불안정 |
| **C. 혼합 모델** ✅ | 기본급 + 성과 인센티브 | 200~670만원 | 균형적 | 복잡성 약간 있음 |
| **D. 대여금 이자** | 2억 × 4.6% ÷ 12 | **77만원** | 세금 단순 | 수익 너무 낮음 |

***

## 9.3 전체 순이익 배분 구조

### 정상 시나리오 (월 순이익 1,246만원 기준)

```
월 순이익 1,246만원 배분
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
장모님 운영 수익   50%  →  623만원
  (약사비 500만원 별도 지급 후 추가 배당)

수진 CFO 보상    32%  →  399만원
  (기본 150만원 + 성과 249만원)

사업 재투자 적립  18%  →  224만원
  (2호점 확장 및 예비 운전자금)

합계            100%  → 1,246만원
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### 사위(송현민) 수익 구조

| 항목 | 내용 |
| ---- | ---- |
| 직접 배당 | 없음 (명의 미포함) |
| 대여금 원금 | 사업 종료·Exit 시 **2억원 우선 회수** |
| 간접 혜택 | 2억원 무이자 → 기회비용 연 920만원 절감 |
| Exit 시 프리미엄 | 3년 후 영업권 매각 시 **7.2억~10.8억** 추정 |

## 9.4 세금 최적화 구조

| 주체 | 소득 형태 | 세율 | 비고 |
| ---- | --------- | ---- | ---- |
| 장모님 | 사업소득 (약국 순이익) | 종합소득세 15% | 급여 500만원 → 필요경비 처리 |
| 수진 | 용역비/경영자문료 | 프리랜서 3.3% 원천징수 | 사업자등록 시 부가세 고려 |
| 사위 | 이자소득 없음 (무이자) | 해당 없음 | 증여세 과세 기준 이하 |

> ℹ️ **절세 팁:** 수진의 경영자문료를 법인 명의로 청구 시 비용처리·4대보험 절감 가능. 세무사 사전 자문 권장.
