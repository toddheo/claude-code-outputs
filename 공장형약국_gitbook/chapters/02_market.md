
| 지표 | 2022~2024 증감 |
| ---- | -------------- |
| 총 매출 증가 | **+3.4%** |
| 매장 수 증가 | **+1.4%** |
| 점포당 매출 증가 | **+2.0%** |

{% hint style="info" %}
매장 수 증가 속도(1.4%) < 매출 증가 속도(3.4%) → **업종 내 구조적 생산성 향상** 진행 중. 신규 진입자에게 유리한 시장 구조.
{% endhint %}

## 2.3 공장형·창고형 약국 성장 타임라인

| ---- | ----------- |

{% endhint %}

## 2.4 공장형 약국의 4대 경쟁 우위

| 우위 요소 | 내용 | 수치 |
| --------- | ---- | ---- |
| **가격** | OTC 대량 매입 할인 | 진통제 20~30% / 외용약 40% / 알레르기약 30% 저렴 |
| **다양성** | 품목 수 압도적 우위 | 5,000개 vs 일반 약국 500개 (10배) |
| **접근성** | 마트 방문객 자연 유입 | 마케팅 비용 Zero CAC |
| **신뢰성** | 약사 상담 + 가격 경쟁력 동시 | "싸지만 믿을 수 있는" 포지셔닝 |

## 2.5 소비자 구매 패턴 변화 (2024~2026)

### 디토(Ditto) 소비

SNS·숏폼 콘텐츠를 통한 약사 추천 제품 품절 현상 확산
→ 마데카솔 분말, 비판텐연고 등 → **입소문 마케팅 비용 절감 가능**

### 최적가 전략

> "싸게 살 때는 온라인, 제대로 고를 때는 약국"

소비자들의 이원적 구매 패턴 정착 → 약국의 **상담 가치 + 가격 경쟁력** 동시 요구

### 경험경제

* 공간 편안함, 편리함, 즐거움이 경영에 직접 영향
* 150평의 넓은 공간 = **브랜드 경험 설계 기회**
