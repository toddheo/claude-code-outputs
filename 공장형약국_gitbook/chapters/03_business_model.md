# 03. 수익 모델 — 처방전 없는 비급여 특화 전략

**\[2026-05-08 전면 개정판]**

***

## 3.1 전략 방향: 왜 처방전을 배제하는가?

```
전통 약국 모델              vs.     공장형 비급여 모델
─────────────────────────────────────────────────────
처방전 조제 의존              →      처방전 완전 배제
병원 트래픽 종속              →      마트 트래픽 + 독립 집객
마진율 15~25% (보험 급여)     →      마진율 40~55%
객단가 10,000~20,000원        →      객단가 45,000~75,000원
일 방문객 상한 있음 (처방 건수)→      상한 없음 (쇼핑형 구매)
─────────────────────────────────────────────────────
```

**처방전 배제의 전략적 근거:**
1. 152평 대형 약국의 강점은 **진열·쇼핑 경험** — 조제실 중심 레이아웃 불필요
2. 비급여약·건기식의 마진율이 처방 급여약 대비 **2~3배 높음**
3. 마트 방문객의 즉흥 구매 패턴 → 처방전 없이도 구매 결정 용이
4. 의사·병원 의존 없이 **독립적 영업 전략** 수립 가능

## 3.2 매출 믹스 전략

| 카테고리 | 비중 | 핵심 품목 | 약국 마진율 | 커미션 | **순마진율** |
| -------- | ---- | --------- | :----------: | :------: | :-----------: |
| **OTC 일반의약품** | **40%** | 감기약·해열제·소화제·진통제 | 35~40% | 없음 | **35~40%** |
| **건강기능식품** | **30%** | 오메가3·유산균·비타민D·마그네슘 | 45~55% | 없음 | **45~55%** |
| **비급여약** | **25%** | 비만약·탈모약·피부약·발기부전약 | 50~60% | 마진의 20% | **40~48%** |
| **반려동물 의약품** | **5%** | 관절·면역·피부 보조제 | 40~50% | 없음 | **40~50%** |

### 혼합 평균 순마진율 계산

```
혼합 마진율 산출
────────────────────────────────────────────────
OTC (40%) × 순마진 37.5%      =  15.0%
건기식 (30%) × 순마진 50%     =  15.0%
비급여약 (25%) × 순마진 44%   =  11.0%
반려동물 (5%) × 순마진 45%    =   2.25%
────────────────────────────────────────────────
혼합 평균 순마진율             ≈  43.25%  →  43%
────────────────────────────────────────────────
```

> ✅ 처방전 없는 모델임에도 일반 처방조제 모델(순마진 38~42%)보다 **마진율 우위**.

## 3.3 비급여약 마진 구조 상세

### 비급여 제휴 모델 개요

```
비급여 제휴 구조
─────────────────────────────────────────────────────
제휴 병원 (비만클리닉·탈모클리닉·피부과)
    ↓ 처방·추천
약국 (판매·조제)
    ↓ 마케팅 협력비 지급
제휴 병원
─────────────────────────────────────────────────────
```

### 비급여약 1만원 매출 기준 마진 흐름

```
비급여약 1만원 매출 기준
────────────────────────────────────────────────────
매출액                         10,000원 (100%)
원가 (도매 매입가)               4,500원  (45%)
────────────────────────────────────────────────────
약국 그로스 마진                  5,500원  (55%)
  ├ 제휴 병원 커미션 (마진의 20%)  1,100원  (11%)
  └ 약국 순마진                   4,400원  (44%)
────────────────────────────────────────────────────
```

> ✅ 비급여 순마진 44%는 OTC(37.5%) 대비 **+6.5%p 우위**. 비급여 비중이 높을수록 전체 수익성 개선.

### 주요 비급여 품목별 마진 구조

| 품목 | 매출 단가 | 원가율 | 그로스 마진율 | 커미션 (11%) | 약국 순마진 |
| ---- | --------- | ------ | ------------- | ------------- | ----------- |
| GLP-1 비만약 (삭센다·위고비 제네릭) | 15,000~50,000원/앰플 | 45% | 55% | 11% | **44%** |
| 탈모약 (피나스테리드·두타스테리드) | 5,000~15,000원/정 | 40% | 60% | 12% | **48%** |
| 미녹시딜 외용액 | 20,000~60,000원/박스 | 42% | 58% | 12% | **46%** |
| 피부 비급여 (트레티노인·하이드로퀴논) | 10,000~30,000원 | 45% | 55% | 11% | **44%** |
| 발기부전약 (실데나필 제네릭) | 5,000~20,000원/정 | 40% | 60% | 12% | **48%** |

> ⚠️ **법적 주의:** 의료기관에 커미션을 지급하는 구조는 의료법·약사법상 리베이트 규정과 저촉될 수 있음. **반드시 의료 전문 변호사 자문을 통해 "마케팅 협력비" 등 적법한 계약 형태로 구성할 것.** → Chapter 06 법적 고려사항 참조.

## 3.5 건강기능식품(건기식) 전략

### 건기식의 전략적 중요성

건기식은 약국 매출 믹스에서 **마진율이 가장 높고(45~55%)**, 상담 연계 판매로 **객단가 인상 효과**가 크다.

| 전략 | 내용 |
| ---- | ---- |
| **약사 전문 상담 판매** | 단순 진열이 아닌 1:1 약사 상담 후 최적 제품 추천 → 객단가 3~5배 상승 |
| **정기구독 모델** | 월 구독형 건기식 패키지 (오메가3+유산균+비타민D) → 재방문율 향상 |
| **독점 브랜드 입점** | 경쟁 약국이 취급 못하는 수입 건기식 독점 공급 협상 → 차별화 |
| **시즌 마케팅** | 봄(알레르기), 여름(다이어트), 가을(면역), 겨울(감기) 테마 기획전 |

### 건기식 정기구독 패키지 설계

| 패키지명 | 구성 | 월 구독가 | 마진 |
| -------- | ---- | --------- | ---- |
| 기본 건강 | 오메가3 + 비타민D + 유산균 | 49,000원 | ~58% |
| 다이어트 케어 | 가르시니아 + L-카르니틴 + 식욕억제 건기식 | 89,000원 | ~54% |
| 탈모 케어 | 판토가 + 미녹시딜 외용액 + 두피 영양제 | 120,000원 | ~46% |
| 반려동물 | 강아지/고양이 관절·면역 영양제 | 39,000원 | ~54% |

## 3.6 반려동물 의약품 전략

### 틈새 시장 공략

2026년 기준 국내 반려동물 양육 가구 약 600만 세대. 반려동물 전용 의약품·건강기능식품은 **약국 정품 구매 선호도가 높고** 고충성 고객군을 형성한다.

| 특징 | 내용 |
| ---- | ---- |
| 반복 구매율 | 월 1~2회 정기 구매 (약품 특성상) |
| 객단가 | 3~8만원/회 |
| 마진율 | 40~50% (건기식 대비 낮지만 안정적) |
| 경쟁 강도 | 일반 약국 미취급률 높음 → 차별화 포인트 |

> ✅ **공간 전략:** 약국 내 '반려동물 존' 별도 구획(15~20평) 설치, 전용 냉장 케이스·진열대 배치 → 반려인 고객 전용 동선 확보.
