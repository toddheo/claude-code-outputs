## 4.2 임대료 협상 전략 및 타당성 분석

### 현재 협상 조건

| 항목 | 건물주 제시 | 협상 목표 |

| **Rent-free** | 없음 | **2~3개월 요청** |

### 협상 타당성 분석

**① 보증금 감액 (2억 → 1~1.5억)**

2026년 현재 서울 구로구 상업용 부동산 시장 동향:
- 공실률 증가 추세 → 임차인 협상력 상승
- 대형 마트 내부 숍인숍 약국: **보증금/보증배율 낮아지는 추세** (마트가 집객 보장 역할)
- 실제 유사 사례: 홈플러스·이마트 내 약국 보증금은 순임대 6~12개월치 → **1,044만 × 12 = 1억 2,528만원** 수준이 합리적

> ✅ **타당성:** 보증금 1억~1.5억원 요청은 **완전히 합리적 범위**. 시장 공실 압박을 활용하면 달성 가능. 신규 마트 입점 초기 킹콩마트 측도 약국 입점을 원하므로 협상 레버리지 존재.

**② Rent-free 2~3개월**

국내 상업용 임대차 시장에서 Rent-free 관행:
- 대형 마트 숍인숍: **인테리어 기간(1~3개월) Rent-free 적용이 일반화**
- 2025~2026년 고금리·공실 환경에서 Rent-free **3개월까지 협상 성공 사례 다수**
- Rent-free 3개월 = 월세 1,500만원 × 3 = **4,500만원 절감** (실질 초기 투자비 27% 감소 효과)

> ✅ **타당성:** Rent-free 2~3개월 요청은 현행 시장 관행에 완전히 부합. **협상 1순위 목표로 설정** 권장 (보증금 감액보다 더 확실한 현금 절감 효과).

**③ 월세 1,500만원 (관리비 포함) 픽스**

| 비교 기준 | 수치 | 판단 |
| --------- | ---- | ---- |
| 순임대 1,044만원 ÷ 152평 | 평당 6.9만원/월 | 서울 구로구 1층 기준 적정 범위 |
| 관리비 456만원 (152평 × 3만원) | 평당 3만원 | 업무·상업 시설 관리비 기준 범위 내 |
| **합계 1,500만원** | 평당 9.9만원/월 | ✅ 마트 숍인숍 기준 적정 수준 |

> ✅ **결론:** 1,500만원 픽스는 현실적 협상 목표. 추가 월세 인하보다 **Rent-free 확보**가 초기 현금흐름 개선에 더 효과적.

## 4.4 손익분기점 (BEP) 분석

### BEP 계산

```
BEP 매출 = 월 고정비 ÷ 혼합 순마진율
         = 4,200만원 ÷ 43%
         = 9,767만원 ≈ 9,800만원 / 월

BEP 일 고객 수 (객단가 55,000원 기준)
         = 9,800만원 ÷ 26일 ÷ 55,000원
         = 약 69명 / 일
```

> ✅ **BEP 의미:** 처방전 없는 비급여 모델의 고마진 효과로 기존 처방 모델 대비 BEP가 대폭 낮아짐. 일 69명이라는 목표는 소형 약국(일 50~100명)도 달성하는 수준.

## 4.7 허진영(사위) 투자 회수 구조

| 구분 | 내용 |
| ---- | ---- |
| 투자 형태 | 대여금 (금전소비대차) 1억~1.5억원, 무이자 |
