## 1.3 수익 모델 개요 (처방전 완전 배제)

```
수익 구조 핵심
────────────────────────────────────────────────────────
처방전 조제 → 0% (완전 배제)

[1] OTC 일반의약품   40%  │  마진율 35~40%  │ 박리다매
[2] 건강기능식품     30%  │  마진율 45~55%  │ 고마진 상담판매
[3] 비급여약         25%  │  순마진율 44%   │ 비만·탈모·피부
[4] 반려동물 의약품   5%  │  마진율 40~50%  │ 틈새 고충성

혼합 평균 순마진율: 43%
────────────────────────────────────────────────────────
```

> ✅ **비급여 커미션 구조:** 약국 마진 50~60% → 제휴 병원 커미션(마진의 20%) 차감 → **약국 순마진 44%**
> 처방전 없는 모델임에도 기존 처방전 포함 모델(39~42%) 대비 마진율 우위.

## 1.6 투자 안전 구조 (4중 레이어)

```
Layer 1: 금전소비대차 공정증서     → 원금 반환 청구권 (허진영)
Layer 2: 임차보증금 반환채권 질권  → 보증금 우선 수취권
Layer 3: 동업계약서 거부권·Exit    → 운영 통제권 (3개월 연속 적자 시 청산 개시)
Layer 4: 사업중단보험 + 배상책임   → 손실 보전
```
