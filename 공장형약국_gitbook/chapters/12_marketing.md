## 12.1 마케팅 전략 개요

공장형 약국의 핵심 마케팅 과제는 두 가지다: **마트 방문객을 약국으로 전환**하는 인바운드 전략과, **마트 방문객 외 독립 집객**을 만드는 아웃바운드 전략.

```
마케팅 전략 2축
────────────────────────────────────────────────────
[축 1] 마트 방문객 전환 최대화
        → 매장 내 시각·후각 자극 + POP 마케팅
        → 마트 공동 프로모션 이벤트
        → 목표: 전환율 20% → 25% 이상으로 끌어올리기

[축 2] 비급여 독립 집객 구축
        → SNS + 네이버 지역 검색 광고
        → 비급여 상담 예약 시스템 운영
        → 제휴 병원 처방→약국 연계
        → 목표: 일 방문객 중 비급여 독립 집객 30% 이상
────────────────────────────────────────────────────
```

## 12.2 오픈 마케팅 (M-0 ~ M+3)

### 오픈 이벤트 전략

| 이벤트 | 내용 | 기대 효과 | 비용 |
| ------ | ---- | --------- | ---- |
| **균일가 이벤트** | 인기 OTC·비타민 5,000원 균일가 (3일간) | 초기 유입 극대화, SNS 입소문 | 50~100만원 (재고 할인 기회비용) |
| **건기식 시음 부스** | 국산 건기식 브랜드 협찬 시음 이벤트 | 건기식 카테고리 인지도 확보 | 무료 (브랜드 협찬) |
| **비급여 무료 상담** | 비만·탈모 1:1 무료 상담 (예약제, 10명/일) | 비급여 잠재 고객 발굴 | 0원 (장모님 상담 시간 활용) |
| **반려동물 존 오픈 기념** | 반려동물 간식 증정 (방문객 한정) | 반려인 고객 유입 | 20~30만원 |
| **마트 공동 이벤트** | 킹콩마트 구매 시 약국 쿠폰 증정 (역방향도 가능) | 마트↔약국 상호 트래픽 강화 | 협찬 협의 |

> ✅ **국내 사례:** 성남 메가팩토리약국 오픈 3개월 만에 일 방문객 200명 돌파. 핵심 요인: 오픈 이벤트 SNS 바이럴 + 인근 병원 제휴 사전 체결.

## 12.4 비급여 특화 마케팅

### 비만·탈모 클리닉 제휴 마케팅

```
비급여 제휴 마케팅 플로우
─────────────────────────────────────────────────
제휴 병원 진료 → 의사 처방 → 약국 조제·판매
       ↓                          ↓
  병원 환자에게               약국 독립 예약
  약국 쿠폰 제공              고객에게 병원 추천
─────────────────────────────────────────────────
```

| 항목 | 내용 |
| ---- | ---- |
| **제휴 병원 리퍼럴 마케팅** | 제휴 병원 대기실에 약국 안내 팸플릿·QR코드 배치 |
| **약국 → 병원 추천** | 비만·탈모 상담 고객에게 제휴 병원 연결 (양방향 리퍼럴) |
| **전용 상담 예약 시스템** | 네이버 예약 또는 카카오 예약 연동 (비급여 상담 예약제 운영) |
| **패키지 상품화** | 비만약 3개월 패키지 + 영양 상담 묶음 상품 |

> ⚠️ **법적 주의:** 의료기관과의 커미션·리베이트는 약사법·의료법 위반 가능성. 마케팅 협력비 명목의 계약은 반드시 **의료 전문 변호사 자문** 후 적법한 형태로 구성.

### 예약제 운영 (비급여 상담)

| 항목 | 내용 |
| ---- | ---- |
| 예약 채널 | 네이버 예약 (무료) + 카카오 채널 |
| 상담 시간 | 평일 오전 10시~12시, 오후 2시~4시 (1일 10명 한정) |
| 상담 내용 | 비만약(GLP-1·식욕억제제) + 탈모약(피나스테리드·미녹시딜) + 피부약 |
| 고객 데이터 | 상담 이력 ERP 등록 → 재방문·재구매 CRM 활용 |

## 12.7 개업 후 월별 마케팅 로드맵

| 월차 | 핵심 마케팅 활동 | 목표 |
| ---- | ---------------- | ---- |
| **M+0~1** | 오픈 이벤트 + 네이버 플레이스 등록 + 인스타그램 개설 | 일 100명 유입, 온라인 인지도 확보 |
| **M+2~3** | 네이버 검색 광고 가동 + 건기식 정기구독 멤버십 론칭 | 재방문율 30% 달성 |
| **M+3~6** | 비급여 예약 상담 시스템 최적화 + 제휴 병원 공동 마케팅 | 비급여 독립 집객 일 20명+ |
| **M+6~12** | 멤버십 VIP 등급 운영 + 분기별 시즌 이벤트 (독감·다이어트 시즌 등) | 회원 500명 확보, 객단가 상승 |
| **M+12+** | 브랜딩 강화 (약국 고유 브랜드 스토리) + 2호점 사전 마케팅 검토 | 지역 내 1위 약국 인지도 확립 |
