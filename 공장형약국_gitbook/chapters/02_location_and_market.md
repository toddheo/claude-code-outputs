## 2.2 킹콩마트 구로점 분석

### 킹콩마트 브랜드 현황

킹콩마트(KingKong Mart)는 2020년대 중반 급성장한 **회원제·창고형 식자재 마트** 모델로, 대형 식자재와 가공식품을 저가·대용량으로 공급하는 방식을 취한다. 구로점은 **기존 세계로마트가 점유하던 공간을 분할 재계약하여 입점**한 것으로, 이전 세계로마트의 집객 인프라(주차장, 접근성, 상권 인지도)를 그대로 승계한다는 점이 강점이다.

| 구분 | 세계로마트 (前) | 킹콩마트 (現, 2026.04.07~) |
| ---- | -------------- | -------------------------- |
| 전체 영업 면적 | ~1,500평 추정 | ~900평 (약국 구획 제외 시) |
| 타겟 고객 | 일반 소비자, 중소 자영업자 | 가족·주부·소규모 자영업자 |
| 집객 방식 | 식자재 품목 다양성 | 저가·대용량·회원 할인 |
| 브랜드 파워 | 지역 인지도 (기성) | 전국 신규 확산 중 |

### 신규 오픈 효과 분석

**긍정 시나리오 (장모님 가설 지지):**
> 신규 마트 오픈 시 첫 3~6개월은 오픈 마케팅(전단·SNS·지역 광고)으로 평시 대비 **120~150% 방문객** 유입이 일반적. 이 기간 약국도 동반 유입 효과 수혜 가능.

**리스크 시나리오:**
> 킹콩마트는 2026년 4월 기준 신규 브랜드로 구로 상권 내 아직 인지도 형성 단계. 안정적 방문객 확보까지 **6~12개월 소요** 예상. 초기 BEP 달성이 지연될 수 있음.

> ⚠️ **핵심 판단 기준:** 개업 전 최소 3개월(2026년 7월~9월 기준) 실제 마트 방문객 카운팅 데이터 확보 필수. 이것이 투자 Go/Stop의 결정적 변수.

## 2.3 마트 일일 방문객 산출 모델

### 모델 A: 면적·유형 기반 추정

국내 식자재 마트·창고형 마트의 방문객 밀도 데이터를 기반으로 한 면적당 추정:

| 마트 유형 | 면적 기준 | 평일 일 방문객 | 주말 일 방문객 | 월 평균 환산 |
| --------- | --------- | ------------- | ------------- | ------------ |
| 대형 (2,000평+) | 코스트코·이마트 | 3,000~8,000명 | 5,000~10,000명 | — |
| 중대형 (800~2,000평) | 킹콩·메가마트 | **500~900명** | **800~1,200명** | **600~950명** |
| 중소형 (300~800평) | 지역 식자재마트 | 200~500명 | 400~700명 | 280~550명 |
| 소형 (~300평) | 동네 마트 | 100~300명 | 200~500명 | 130~350명 |

> 킹콩마트 구로점(~900평 추정) → **평일 평균 400~700명, 주말 600~1,000명** → **월 평균 500~750명/일** 예상

### 모델 B: 유사 상권 벤치마크

| 사례 | 마트 유형 | 마트 면적 | 일 방문객 | 약국 전환율 | 약국 일 방문객 |
| ---- | --------- | --------- | --------- | ----------- | -------------- |
| 홈플러스 금천점 (메가팩토리 입점) | 대형 | 2,000평+ | 3,000명+ | 5~8% | 150~240명 |
| 이마트 성수점 (대형 약국 입점) | 대형 | 3,000평 | 5,000명 | 4~6% | 200~300명 |
| **킹콩마트형 중대형 마트** | **중대형** | **~900평** | **500~750명** | **15~25%** | **75~188명** |
| 지역 중형 식자재 마트 | 중형 | ~500평 | 300~500명 | 20~30% | 60~150명 |

> ✅ **결론:** 킹콩마트 구로점 기준 일 방문객 **500~750명** 시나리오에서, 약국 전환율 **20%** 적용 시 약국 일 방문객 **100~150명** 달성 가능.

### 전환율(Conversion Rate) 산출 근거

약국 전환율은 마트 방문객 중 약국에 실제 입장하는 비율이다. 이를 결정하는 핵심 변수:

```
전환율 결정 변수
─────────────────────────────────────────────────────
(+) 마트 입구 또는 주동선 인접 배치 → +5~10%p
(+) 대형 간판·눈에 띄는 파사드 설계 → +3~5%p
(+) 오픈 이벤트·균일가 프로모션    → +5~8%p (초기 3개월)
(+) 비급여 상담 예약제 운영         → +2~4%p (독립 집객)
(-) 마트 내 동선 깊은 곳에 위치     → -5~10%p
(-) 경쟁 약국 인근 존재             → -3~5%p
─────────────────────────────────────────────────────
기준 전환율 (중립 입지): 15~20%
최적 조건 달성 시: 20~28%
```

## 2.5 장모님 가설 (일 200~300명 / 객단가 5~10만원) 타당성 검증

### 검증 결과: 조건부 타당

| 조건 | 필요 전제 | 현실 가능성 |
| ---- | --------- | ----------- |
| **일 200명** | 마트 일 방문객 1,000명 이상 + 전환율 20% | 마트 성숙기(1년 이상) 도달 시 가능 |
| **일 300명** | 마트 일 방문객 1,200명+ + 비급여 독립 집객 50명 | 최우수 시나리오, 2~3년 차 |
| **객단가 7만원** | 비급여약 비중 35%+ 달성 | 제휴 병원 3곳 이상 확보 필요 |
| **객단가 10만원** | 비급여 50%+ + 고가 건기식 세트 판매 | 장기 안정기 이후 현실적 |

```
시나리오별 월 매출 예측 (일 방문객 × 객단가 × 26일)
────────────────────────────────────────────────────────
정상 시나리오  : 150명 × 55,000원 × 26일 = 2억 1,450만원
장모님 예상치  : 200명 × 75,000원 × 26일 = 3억 9,000만원
최우수 시나리오: 250명 × 70,000원 × 26일 = 4억 5,500만원
────────────────────────────────────────────────────────
```

> ⚠️ **중요:** 장모님의 200~300명/일 예상은 비급여 제휴 병원 다수 확보 + 마트 방문객 충분한 안정화가 전제될 경우에만 현실적. **개업 첫 해 목표는 보수적으로 일 80~100명으로 설정**하고 단계적으로 확장하는 것이 리스크 관리 측면에서 합리적.
