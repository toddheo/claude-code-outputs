# 8. 법적 보호 구조

**투자금 2억원 안전 설계 — 4중 레이어**

***

## 보호 구조 개요

```
┌─────────────────────────────────────────┐
│  Layer 1: 금전소비대차계약 (공정증서)      │  원금 반환 청구권
├─────────────────────────────────────────┤
│  Layer 2: 임차보증금 반환채권 질권 설정   │  보증금 우선 수취권
├─────────────────────────────────────────┤
│  Layer 3: 동업계약서 (거부권·Exit 조항)  │  운영 통제권
├─────────────────────────────────────────┤
│  Layer 4: 사업중단보험 + 배상책임보험    │  손실 보전
└─────────────────────────────────────────┘
```

***

## Layer 1 — 금전소비대차계약 (무이자)

**목적:** 사위→장모님 2억원 대여의 법적 근거 확보, 증여세 회피

```
계약 구조
────────────────────────────────
대여자  : 사위 (송현민)
차용자  : 장모님 (약사)
금액    : 2억원
이자    : 무이자
만기    : 사업 종료 시 또는 5년
공증    : 공정증서 (법무사·공증인 공증 필수)
────────────────────────────────
```

### 증여세 리스크 분석

| 항목 | 수치 |
| ---- | ---- |
| 무이자 2억원 → 연 이자 상당액 | 2억 × 4.6% = **920만원/년** |
| 과세 최저한 | **1,000만원** |
| **과세 여부** | **증여세 과세 없음** ✅ |

> ⚠️ 매년 계약 갱신 필요. 향후 기준이자율 상승 시 재검토 필수.

***

## Layer 2 — 임차보증금 반환채권 질권 설정

**목적:** 사위가 보증금 2억원 회수 권리를 법적으로 확보

```
설정 방법
────────────────────────────────────────
질권자       : 사위 (채권자)
질권설정자   : 장모님 (임차인)
목적물       : 킹콩마트와의 임대차계약상
               보증금 반환채권 2억원
필요 조건    : 임대인(킹콩마트) 동의서
               + 내용증명 발송
────────────────────────────────────────
```

> ❌ **주의:** 임대인 동의 없으면 질권 효력이 제3자에게 대항 불가. 임대차계약서에 "보증금 반환 시 사위 계좌로 송금" 조항 삽입 필수.

***

## Layer 3 — 동업계약서 (사실상 투자계약)

### 핵심 조항

| 조항 | 내용 |
| ---- | ---- |
| **투자금 인정** | 사위의 2억원 = 사업의 대출금으로 명시 |
| **원금 우선 보호** | 폐업 시 보증금 반환액 中 사위에게 **우선 변제** |
| **이익 배분** | 순이익의 30~40%를 사위에게 경영자문료·대여금 이자 명목 지급 |
| **거부권 (Veto)** | 월세 외 추가 임대 계약, **500만원 이상 지출** 시 사위 동의 필수 |
| **Exit 조건** | 3개월 연속 적자 시 사위 요청으로 **청산 절차 개시** |
| **정보 공유** | 매출·지출 내역 CFO(수진)에게 **실시간 공유 의무** |

***

## Layer 4 — 보험 설계

| 보험 종류 | 용도 | 월 보험료 추정 |
| --------- | ---- | -------------- |
| 약사 배상책임보험 | 조제 사고 대비 | 5~10만원 |
| 사업장 화재·재물보험 | 약품·집기 손해 대비 | 10~15만원 |
| **사업중단보험** | **대표자 입원 시 고정비 보전** | 20~30만원 |
| 종합 약사 의료배상 | 의약품 오조제 소송 대비 | 포함 |
| **합계** | | **약 35~55만원/월** |

***
