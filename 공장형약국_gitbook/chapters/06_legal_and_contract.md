# 06. 법적 고려사항 및 계약 체계

**\[2026-05-08 전면 개정판]**

## 6.1 투자금 보호 4중 레이어

```
┌─────────────────────────────────────────────┐
│  Layer 1: 금전소비대차계약 (공정증서)          │  원금 반환 청구권
├─────────────────────────────────────────────┤
│  Layer 2: 임차보증금 반환채권 질권 설정       │  보증금 우선 수취권
├─────────────────────────────────────────────┤
│  Layer 3: 동업계약서 (거부권·Exit 조항)      │  운영 통제권
├─────────────────────────────────────────────┤
│  Layer 4: 사업중단보험 + 배상책임보험        │  손실 보전
└─────────────────────────────────────────────┘
```

## 6.2 Layer 1 — 금전소비대차계약 (무이자)

**목적:** 허진영(사위) → 장모님 대여금의 법적 근거 확보, 증여세 회피

```
계약 구조
────────────────────────────────────────────
대여자  : 허진영 (사위)
차용자  : 장모님 (약사, 개인사업자)
금액    : 1억~1.5억원 (협상 결과 확정)
이자    : 무이자
만기    : 사업 종료 시 또는 5년
공증    : 공정증서 필수 (법무사 또는 공증인)
────────────────────────────────────────────
```

### 증여세 리스크 분석

| 항목 | 수치 |
| ---- | ---- |
| 무이자 1억원 → 연 이자 상당액 | 1억 × 4.6% = **460만원/년** |
| 무이자 1.5억원 → 연 이자 상당액 | 1.5억 × 4.6% = **690만원/년** |
| 과세 최저한 | **1,000만원** |
| **과세 여부** | **증여세 과세 없음** ✅ |

> ⚠️ 기준이자율(현행 4.6%) 상승 시 재검토 필요. 매년 계약 갱신 또는 이자율 조항 검토 권장.

***

## 6.3 Layer 2 — 임차보증금 반환채권 질권 설정

**목적:** 허진영이 보증금 우선 회수 권리를 법적으로 확보

```
설정 방법
────────────────────────────────────────────────────
질권자       : 허진영 (채권자)
질권설정자   : 장모님 (임차인)
목적물       : 킹콩마트와의 임대차계약상 보증금 반환채권
필요 조건    : 임대인(킹콩마트) 서면 동의 + 내용증명 발송
────────────────────────────────────────────────────
```

> ❌ **주의:** 임대인 동의 없으면 질권 효력이 제3자에게 대항 불가. 임대차계약서에 "보증금 반환 시 허진영 계좌로 송금" 조항 삽입 필수.

***

## 6.4 Layer 3 — 동업계약서 핵심 조항

| 조항 | 내용 |
| ---- | ---- |
| **투자금 인정** | 허진영의 대여금 = 사업의 실질 투자금으로 명시 |
| **원금 우선 보호** | 폐업 시 보증금 반환액 中 허진영에게 우선 변제 |
| **이익 배분** | 권수진 CFO 인센티브(영업이익 15%) 구조 명시 |
| **거부권 (Veto)** | 월세 외 추가 임대·**500만원 이상 지출** 시 허진영 동의 필수 |
| **Exit 조건** | 3개월 연속 적자 시 허진영 요청으로 **청산 절차 개시** |
| **정보 공유** | 매출·지출 내역 CFO(권수진)에게 **실시간 공유 의무** |

## 6.5 약사(장모님) ↔ 건물주(킹콩마트) 임대차계약 필수 특약

### A. 반드시 삽입해야 할 특약 사항

| 특약 조항 | 내용 | 중요도 |
| --------- | ---- | ------ |
| **마트 폐점 조기 해지권** | "킹콩마트 구로점 영업 중단 시 임차인 즉시 해지 가능, 보증금 전액 반환" | 🔴 최우선 |
| **보증금 반환 계좌 지정** | "계약 종료 시 보증금을 허진영 계좌(계좌번호 기재)로 반환" | 🔴 최우선 |
| **임대인 질권 동의** | "임차인의 보증금 반환채권에 대한 질권 설정을 임대인이 동의함" | 🔴 최우선 |
| **방문객 보장 재협상권** | "마트 3개월 평균 일 방문객 400명 미만 지속 시 임차인 월세 재협상 요청권 부여" | 🟠 중요 |
| **목적 제한** | "본 임대 공간은 약국 영업 목적으로만 사용하며, 임대인은 동종 업종(약국) 추가 임대를 금지한다" | 🟠 중요 |
| **Rent-free 기간** | "인테리어 공사 기간 ___개월간 임대료 면제 (관리비 포함)" | 🟠 중요 |
| **원상복구 범위 한정** | "원상복구 의무는 구조 변경분에 한하며, 일반 소모 및 장치 인테리어는 제외한다" | 🟡 보통 |
| **전대차 금지** | "임차인은 해당 공간을 약국 목적 외로 사용하거나 제3자에게 전대차할 수 없다" | 🟡 보통 |
| **주차 이용권** | "임차인 및 약국 방문 고객에게 마트 주차장 ___대 무상 이용권 부여" | 🟡 보통 |

### B. 임대차 협상 우선순위

```
협상 우선순위
────────────────────────────────────────────────
1순위: Rent-free 2~3개월 확보
  → 현금 3,000~4,500만원 절감, 즉각적 효과

2순위: 마트 폐점 시 조기 해지·보증금 즉시 반환 조항
  → 최대 리스크(마트 폐점) 계약서로 방어

3순위: 임대인 질권 동의 + 보증금 반환 계좌 지정
  → 허진영 원금 보호의 핵심

4순위: 보증금 감액 (2억 → 1~1.5억)
  → 협상 카드 순서 조정, 3순위 후 제시 효과적
────────────────────────────────────────────────
```

## 6.6 약사(장모님) ↔ 권수진 동업(운영대리) 계약

| 항목 | 내용 |
| ---- | ---- |
| **권수진의 역할** | CFO — 재무·회계·법무·마케팅·인사 총괄 |
| **기본 보상** | 월 250만원 고정 경영자문료 |
| **성과 인센티브** | 영업이익의 15% (분기별 정산) |
| **의사결정권** | 500만원 이상 지출 승인권, 인사 결정 참여 |
| **정보 접근권** | 모든 회계·매출·지출 내역 실시간 접근 |
| **계약 기간** | 사업 지속 기간 (최소 1년 단위 갱신) |

### 권수진 CFO 영입 사전 준비 체크리스트

| 항목 | 내용 | 우선순위 | 기한 |
| ---- | ---- | :------: | ---- |
| **사업자등록** | 경영자문 용역 세금계산서 발행용 개인사업자 등록 | 🔴 최우선 | 개업 전 |
| **세무사 자문** | 용역소득 vs 근로소득 최적화 (3.3% 프리랜서 vs 근로계약) | 🔴 최우선 | 개업 전 |
| **동업계약서 체결** | 장모님-권수진-허진영 3자 계약서 (변호사 작성) | 🔴 최우선 | 개업 전 |
| **약국 전산 접근 권한** | POS·ERP·회계 시스템 권수진 관리자 접근 설정 | 🟠 긴급 | 개업 시 |
| **법인 신용카드 발급** | 약국 사업 경비 전용 카드 | 🟠 긴급 | 개업 시 |
| **재무 보고 체계 구축** | 허진영에게 월 1회 재무 보고 루틴 (매월 5일) | 🟡 중요 | 개업 1개월 내 |
| **인센티브 정산 기준 문서화** | 영업이익 계산 방식·정산 시기·방법 합의 서면화 | 🟡 중요 | 개업 전 |
| **근로계약서 작성** | 관리인원 3명 + 파트타임 약사 계약서 | 🟡 중요 | 개업 전 |

## 6.7 약국 ↔ 비급여 제휴 병원 계약

### 계약 구조 및 핵심 조항

| 항목 | 내용 |
| ---- | ---- |
| **제휴 대상** | 비만클리닉·탈모클리닉·피부과 등 비급여 전문 의원 |
| **제휴 방식** | 병원 처방→약국 조제·판매 연계 + 양방향 리퍼럴 |
| **커미션 구조** | 약국 마진의 20%를 "마케팅 협력비" 명목으로 제휴 병원에 지급 |
| **독점 조항** | 반경 300m 내 동일 품목 타약국 제휴 금지 협상 시도 |
| **계약 기간** | 6개월 단위 갱신 |
| **리베이트 방지** | 의료법·약사법 리베이트 규정 준수 명시, 영수증·세금계산서 발급 의무화 |

### ⚠️ 법적 리스크 및 방지 장치

> ❌ **핵심 주의사항:** 의료기관·약국 간 경제적 이익 수수는 의료법 제23조의2(리베이트 수수 금지)·약사법 제47조의2의 규제 대상이 될 수 있다. 아래 사항을 반드시 준수할 것.

| 리스크 | 방지 방법 |
| ------ | --------- |
| 처방 유도 대가 성격의 커미션 | "마케팅 협력비" 명목, 실제 공동 마케팅 활동(팜플릿·SNS 공동 광고) 증빙 첨부 |
| 현금성 리베이트 의심 | 반드시 세금계산서 발행 + 계좌이체 기록 유지 |
| 처방 내용 개입 | 약국이 특정 의약품 처방 요청하는 것 금지 |
| 계약 형태 | 의료 전문 변호사·법무사 자문 후 적법한 형태 확정 |

> ⚠️ **강력 권고:** 비급여 제휴 계약 체결 전 **의료·약사법 전문 변호사 1회 이상 자문 필수**.

## 6.8 약국 ↔ 도매상 공급 계약

| 항목 | 내용 |
| ---- | ---- |
| **주요 도매상** | 지오영, 백제약품, 인정약품 등 2~3곳 동시 계약 |
| **결제 조건** | Net 30~60일 (초기 현금 흐름 확보에 핵심) |
| **가격 조건** | 대량 매입 단가 할인 10~20% 목표 (연간 매입 규모 협상) |
| **반품 조건** | 유통기한 3개월 이전 반품 가능 조항 삽입 |
| **독점 공급** | 특정 건기식 브랜드 지역 독점 공급 협상 |

## 6.9 세금 최적화 구조

| 주체 | 소득 형태 | 세율 | 절세 포인트 |
| ---- | --------- | ---- | ----------- |
| **장모님** | 사업소득 (약국) | 종합소득세 15~35% | 장모님 급여 500만원 → 필요경비 처리로 과세표준 절감 |
| **권수진** | 용역소득 (경영자문료) | 3.3% 원천징수 후 종합소득세 신고 | 사업자등록 시 VAT 매입공제 + 필요경비 처리 가능 |
| **허진영** | 이자소득 없음 (무이자 대여) | 해당 없음 | 증여세 과세 기준(연 460~690만원) 이하, 과세 없음 |

## 6.10 보험 설계 (Layer 4)

| 보험 종류 | 용도 | 월 보험료 추정 |
| --------- | ---- | -------------- |
| **약사 배상책임보험** | 조제·상담 사고 법적 대응 | 5~10만원 |
| **사업장 화재·재물보험** | 약품·집기 손해 보전 | 10~15만원 |
| **사업중단보험** | 장모님 입원 시 고정비(4,200만원/월) 보전 | 20~30만원 |
| **종합 약사 의료배상** | 약품 관련 소송 대응 | 포함 |
| **합계** | | **약 35~55만원/월** |
