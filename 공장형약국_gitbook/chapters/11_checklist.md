# 11. 투자 의사결정 체크리스트

***

## ✅ Go 조건 — 모두 충족 시 투자 권장

{% hint style="success" %}
아래 8개 항목을 **모두 체크**한 후 투자를 진행하세요.
{% endhint %}

* [ ] 임대차계약서 법률 검토 완료 (임대인 동의 조항 삽입 확인)
* [ ] 금전소비대차 **공정증서 완료** (보증금 2억 법적 보호)
* [ ] 동업계약서 체결 완료 (Exit 조항·거부권 명시)
* [ ] 파트타임 약사 후보 **Pool 2인 이상** 확보
* [ ] 인근 약국 경쟁 현황 실사 완료 (반경 500m 직접 조사)
* [ ] 킹콩마트 **일일 방문객 수 현장 카운팅** 완료 (675명+ 확인)
* [ ] 인테리어 비용 5,000만원 조달 계획 확정
* [ ] 인근 의원 **1~2곳 사전 관계 구축** 완료

***

## ❌ Stop 조건 — 하나라도 해당 시 재검토

{% hint style="danger" %}
아래 중 **하나라도 해당**하면 투자를 중단하고 재검토하세요.
{% endhint %}

* [ ] 월세 재협상 실패 (1,500만원 고정 불변 + 추가 인상 조항 존재)
* [ ] 반경 300m 내 창고형 약국 **기출점** 확인
* [ ] 킹콩마트 일일 방문객 **300명 미만** 으로 추정
* [ ] 장모님 건강 상태 불확실 (대체 약사 Pool 미확보)
* [ ] 임대인이 **질권 설정 동의 거부**

***

## 투자 판단 매트릭스

| 상황 | 판단 |
| ---- | ---- |
| Go 조건 8개 모두 충족 + Stop 조건 0개 | ✅ **적극 투자 권장** |
| Go 조건 6~7개 충족 + Stop 조건 0개 | ⚠️ 미충족 항목 보완 후 재판단 |
| Go 조건 5개 이하 OR Stop 조건 1개 이상 | ❌ **투자 재검토 필요** |
