## ✅ Go 조건 — 모두 충족 시 투자 권장

> ✅ 아래 **10개 항목을 모두 체크**한 후 투자를 진행하세요.

* [ ] 킹콩마트 **일일 방문객 현장 카운팅 완료** — 평일+주말 5일 기준 400명 이상 확인
* [ ] 반경 500m 경쟁 약국 현황 실사 완료 (창고형·메가팩토리 미출점 확인)
* [ ] 임대차계약서 법률 검토 완료 (질권 동의·마트폐점 해지권·Rent-free 조항 삽입 확인)
* [ ] 금전소비대차 **공정증서 완료** (보증금 1~1.5억원 법적 보호, 허진영 명의)
* [ ] 동업계약서 체결 완료 (Exit 조항·거부권·수진 CFO 보상 명시)
* [ ] 임차보증금 반환채권 **질권설정 완료** (임대인 동의서 수령)
* [ ] **비급여 제휴 병원 1곳 이상 사전 협의** 완료 (비만·탈모·피부과 중 1곳)
* [ ] 파트타임 약사 후보 **Pool 2인 이상** 확보 (긴급 대체 가능)
* [ ] 수진 사업자등록 완료 + 동업계약서 날인
* [ ] 인테리어 비용 **5,500만원 이하** 견적 확보 (Rent-free 2~3개월 내 완공 가능한 일정)

## ❌ Stop 조건 — 하나라도 해당 시 재검토

> ❌ 아래 중 **하나라도 해당**하면 투자를 중단하고 재검토하세요.

* [ ] 킹콩마트 일일 방문객 **300명 미만**으로 추정 (3개월 카운팅 평균)
* [ ] 반경 300m 내 창고형·메가팩토리 약국 **기출점** 또는 **출점 예정** 확인
* [ ] 임대인이 **질권 설정 동의 거부** (보증금 반환채권 보호 불가)
* [ ] 월세 재협상 실패 (1,500만원 초과 + Rent-free 거부)
* [ ] 비급여 제휴 병원 **사전 접촉 전면 거부** (의료법 리베이트 우려 포함)
* [ ] 장모님 건강 상태 불확실 또는 대체 약사 Pool **미확보**
* [ ] 킹콩마트 영업 안정성 의심 신호 (모 기업 재무 이슈, 구로점 계약 불명확)
