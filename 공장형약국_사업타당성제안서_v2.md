### 2.2 공장형 약국이란?

2025년 하반기부터 전국적으로 확산된 **창고형·공장형 약국** 트렌드:

- 기존 약국 대비 3~10배 넓은 면적
- SKU(취급 품목) 500개 → 3,000~5,000개로 대폭 확대
- 가격 경쟁력: OTC 일반의약품 **10~60% 저렴** (대량 매입·박리다매 전략)
- 건강기능식품·뷰티·반려동물 섹션 병행 운영
- 집객 앵커(Anchor Tenant)로서 마트·쇼핑몰 유치 대상

**산업 레퍼런스:**
| 매장 | 위치 | 규모 | 특이사항 |
|------|------|------|----------|
| 메가팩토리약국 금천점 | 홈플러스 금천점 내 | 870평 (총 1,740평) | 약사 10명, 품목 5,000+ |
| 메가팩토리약국 성남점 | 성남 | 130평 | 1호점, 2025.06 오픈 |
| 롯데마트 맥스 입점 약국 | 경남 창원 | 대형 | 숍인숍 |
| 스타필드 빌리지 운정 | 경기 파주 | 대형 | 숍인숍 |

→ **킹콩식자재마트 150평 입점**은 이 트렌드의 **적절한 중간 규모** (성남점 130평 대비 약간 大, 금천점 870평 대비 소형)

## 3. 시장 환경 분석 (실데이터 기반)

### 3.1 전국 약국 산업 현황

**[출처: 카드매출 실데이터 / 건강보험심사평가원 / 한국보건의료원]**

| 지표 | 수치 | 비고 |
|------|------|------|
| 전국 약국 수 | 25,062개 | 2024년 기준 |
| 전국 약국 평균 연매출 | 약 12억 3,620만원 | 카드매출 집계 기준 |
| 전국 약국 평균 월매출 | 약 1억 300만원 | 위 연매출 ÷ 12 |
| 연평균 매출 성장률 | **7.4%** | 2019~2024, 신용카드사 집계 |
| 약국당 월평균 조제료 | **1,431만원** | 처방조제 요양급여비 기준 (2021) |
| 약국당 연간 요양급여비 | 1억 7,173만원 | 처방전 조제 수입 기준 |
| 총 약국 요양급여비 | 4조 825억원 | 전국 합산 |

**시사점:**
- 전국 평균 월매출 1억 300만원 → 목표 1억 3,000만원은 **상위 30~40%** 수준
- 강남역 약국 카드매출 중간값 6,387만원 → 구로구 위치 고려 시 **8,000~1억원** 도달 가능한 현실적 목표
- 처방조제만으로 월 1,431만원 → OTC·건기식 추가 시 구조적 매출 상향 가능

### 3.2 업종 성장성

```
약국 업종 카드매출 추이 (2019~2024)
────────────────────────────────────

2024  ▓▓▓▓▓▓▓▓▓▓▓▓▓    연평균 +7.4% 달성
────────────────────────────────────
- 매장 수 증가: +1.4% / 매출 증가: +3.4% / 점포당 매출 증가: +2%

```

### 3.3 공장형·창고형 약국 트렌드

| 연도 | 주요 이벤트 |
|------|-------------|
| 2025.06 | 성남 메가팩토리약국 1호점 오픈 (130평) |
| 2026.02 | 메가팩토리약국 서울 금천점 오픈 (870평, 홈플러스 내) |
| 2026.04 | 전국 창고형 약국 허가 50개 이상 달성 |
| 2026.05 | 킹콩식자재마트 150평 약국 (현재 검토 中) |

**주요 경쟁 우위 요인:**
1. **가격**: OTC 10~60% 저렴 (진통제 20~30%, 외용약 40%, 알레르기약 30%)
2. **다양성**: 품목 3,000~5,000개 취급 (일반 약국 500개 대비 6~10배)
3. **접근성**: 마트 방문객 자연 유입 (Zero 마케팅 비용)
4. **신뢰성**: 약사 상담 + 대량 매입 = 가격 + 전문성 동시 충족

### 3.4 소비자 구매 패턴

**2024년 약국 트렌드 (데일리팜):**
- **디토 소비**: SNS·숏폼 콘텐츠를 통한 약사 추천 제품 품절 현상 (마데카솔 분말, 비판텐연고 등)
- **최적가 전략**: "싸게 살 때는 온라인, 제대로 고를 때는 약국" — 약국의 상담 가치 유지
- **경험경제**: 공간 편안함, 편리함, 즐거움이 경영에 직접적 영향

## 7. 손익분기점 (BEP) 분석

### 7.1 BEP 공식

```
BEP 매출 = 월 고정비 ÷ 마진율
         = 3,500만원 ÷ 40%
         = 8,750만원 / 월
```

### 7.2 BEP 고객 수 역산

```
BEP 일 고객 = BEP 월 매출 ÷ 영업일 ÷ 객단가
            = 8,750만원 ÷ 26일 ÷ 25,000원
            = 약 135명 / 일
```

### 7.3 BEP 도달 시나리오

| 가정 | BEP 매출 | BEP 일 고객 |
|------|----------|-------------|
| 객단가 20,000원 (저가 고객) | 8,750만원 | 168명 |

**전략적 시사점:**
- 객단가 상향이 BEP 달성의 핵심 레버
- 처방전 환자(객단가 35,000원+) 유입 시 **일 96명**으로 BEP 달성
- 마트 방문 고객 전환율 15% 유지 시 **일 550명 마트 방문객** 필요

### Risk 3 — "대표자 건강 이슈 / 약사 부재" 리스크

| 기간 | 영향 |
|------|------|
| 1~2주 단기 | 파트타임 약사 고용으로 운영 가능 |

**대응 전략:**
1. **파트타임 약사 Pool 사전 확보**: 채용 플랫폼(히든랩, 잡코리아 약사 채용) 통해 2~3인 후보 대기 명단 유지
2. **약사 채용보험 가입**: 의료인 사업중단보험 검토
3. **대체 약사 파트너 계약**: 인근 약대 출신 예비약사와 파트타임 고용계약 선체결
4. **법적 구조**: 장모님 유고 시 영업허가 명의이전 절차 사전 파악 (보건소 상담 필수)

**법적 보호 한계:** 약사면허 명의 이전은 불가. 가족 중 약사 면허자 추가 또는 위탁 약사 확보가 유일한 해결책.

## 9. 법적 보호 구조 — 투자금 2억원 안전 설계

### 9.1 4중 법적 보호 레이어

```
Layer 1: 금전소비대차계약 (공정증서)
Layer 2: 임차보증금 반환채권 질권 설정
Layer 3: 동업계약서 (수익 배분·거부권·Exit 조항)
Layer 4: 약사업 배상책임보험 + 사업중단보험
```

### Layer 1 — 금전소비대차계약 (무이자)

**목적:** 사위→장모님 2억원 대여의 법적 근거 확보, 증여세 회피

```
계약 구조:
  대여자: 사위 (송현민)
  차용자: 장모님 (약사)
  금액: 2억원
  이자: 무이자
  만기: 사업 종료 시 또는 5년
  공증: 공정증서 (법무사·공증인 공증 필수)
```

**증여세 리스크 분석:**
- 무이자 2억원 → 연 이자 상당액: 2억 × 4.6% = 920만원/년
- 연간 이자 이익 920만원 < 과세 최저한 **1,000만원** → **증여세 과세 없음**
- 단, 매년 갱신 필요. 향후 이자율 상승 시 재검토 필요.

### Layer 2 — 임차보증금 반환채권 질권 설정

**목적:** 장모님 대신 사위가 보증금 2억원 회수 권리 확보

```
설정 방법:
  질권자: 사위 (채권자)
  질권설정자: 장모님 (임차인)
  목적물: 킹콩마트와의 임대차계약상 보증금 반환채권
  필요 조건: 임대인(킹콩마트 측) 동의서 + 내용증명 발송
```

**주의사항:**
- 임대인 동의 없으면 질권 효력 제3자 대항 불가
- 임대인 설득 필요 ("보증금 반환 시 사위 계좌로 송금" 조항 삽입)

### Layer 3 — 동업계약서 (사실상 투자계약)

**핵심 조항:**

| 조항 | 내용 |
|------|------|
| 투자금 인정 | 사위의 2억원 = 사업의 대출금으로 명시 |
| 원금 보호 | 폐업 시 보증금 반환액 中 사위에게 우선 변제 |
| 이익 배분 | 순이익의 30~40%를 사위에게 경영자문료·대여금 이자 명목 지급 |
| 거부권 | 월세 이외 추가 임대 계약, 500만원 이상 지출 시 사위 동의 필수 |
| Exit 조건 | 3개월 연속 적자 시 사위 요청으로 청산 절차 개시 |
| 개인정보 | 매출·지출 내역 CFO(수진)에게 실시간 공유 의무 |

### Layer 4 — 보험 설계
