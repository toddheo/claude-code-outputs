5. GTM(Go-To-Market) 및 군 특화 진입 전략

5. GTM(Go-To-Market) 및 군 특화 진입 전략

5.1 GTM 전략 개요: 3단계 점진적 확산 모델

아임인의 군 시장 진입은 '증명 → 확장 → 제도화' 의 3단계 전략으로 추진합니다. 단기간 내 전군 확산을 시도하는 대신, 파일럿 성과로 신뢰를 쌓고 국방부의 자발적 확산을 유도하는 Bottom-Up 전략을 채택합니다.

Phase 1 (0~6개월)      Phase 2 (7~18개월)     Phase 3 (19~36개월)
  파일럿 실행            지역 확산               전군 제도화
    │                       │                       │
1개 사단               5개 사단               전 육·해·공군
약 1만명               약 5만명               약 36만명
검증 데이터 수집       성과 확산·개선          국방부 공식 복지 등재

5.2 Phase 1: 파일럿 — 증명의 6개월

5.2.1 파일럿 대상 선정 기준

최초 파일럿 부대는 다음 기준으로 선정합니다.

선정 기준 우선순위 이유
스마트폰 사용 자유도 높은 부대 1순위 앱 사용 빈도 극대화
연령대 낮은 장병 비중 높은 부대 2순위 디지털 친화도
단일 주둔지(캠프) 형태 3순위 오프라인 온보딩 용이
국방부 금융교육 시범 부대 지정 여부 4순위 정책 연계 시너지

추천 파일럿 후보: 수도방위사령부 예하 특정 연대 (서울 근교, 접근성 양호)

5.2.2 파일럿 목표 KPI

KPI 목표 측정 방법
가입률 전체 장병의 40% 이상 앱 가입자 수 / 전체 장병 수
월간 활성 사용자(MAU) 가입자의 70% 이상 월 1회 이상 앱 접속
스테이지 완주율 85% 이상 3개월 스테이지 납입 완료 비율
연체율 0.3% 이하 아임인 전체 연체율(0.22%) + 군 리스크 버퍼
고위험 투자 전환율 감소 30% 이상 감소 파일럿 전후 설문 비교
순추천지수(NPS) 60점 이상 장병 만족도 조사

5.2.3 파일럿 6개월 실행 계획

주차 활동 담당
W1~2 국방부 업무협약(MOU) 체결 + 금감원 혁신금융서비스 신청 티웨이브 BD팀
W3~4 파일럿 부대 선정 + 지휘관 브리핑 티웨이브 + 국방부
W5~6 밀리터리 에디션 앱 배포 + 나라사랑카드 연동 설정 티웨이브 기술팀
W7~8 장병 대상 온보딩 교육 (집체 교육 30분) 티웨이브 CS팀
W9~12 첫 스테이지 그룹 매칭 + 운영 안정화 운영팀
W13~20 운영 중 모니터링 + 이슈 개선 기술+운영팀
W21~24 파일럿 결과 분석 + 보고서 작성 BD+데이터팀
W25~26 국방부·금감원 결과 보고 및 Phase 2 승인 요청 CEO + BD팀

5.2.4 파일럿 비용 구조

항목 금액 비고
앱 군 특화 개발 5,000~8,000만원 나라사랑카드 연동 포함
온보딩 교육 운영 500~1,000만원 오프라인 세션 진행비
데이터 보안 강화 1,000~2,000만원 국방부 보안 기준 충족
운영 인건비 (6개월) 3,000~5,000만원 전담 팀 2~3명
합계 9,500~16,000만원 약 1~1.6억원

ℹ️ 비용 분담 제안: 국방부가 파일럿 비용의 30~50%를 '금융 교육 예산'으로 지원하는 공동 투자 구조를 제안합니다. 이는 국방부가 사업 결과에 공동 책임을 갖는 구조로, 진정한 민관 파트너십의 기반이 됩니다.

5.3 Phase 2: 지역 확산 — 5개 사단으로 성장

5.3.1 Phase 2 진입 기준 (Gate Review)

Phase 1 완료 후 다음 조건 충족 시 Phase 2 자동 진행:

진입 조건 기준값 실제 달성 여부
파일럿 가입률 40% 이상 Phase 1 완료 후 확인
연체율 0.3% 이하 Phase 1 완료 후 확인
금감원 혁신금융서비스 지정 완료 Phase 1 기간 내 진행
국방부 내부 승인 완료 Phase 1 기간 내 진행

5.3.2 Phase 2 목표

지표 Phase 1 달성 Phase 2 목표
적용 부대 수 1개 사단 5개 사단
장병 사용자 수 ~4,000명 ~20,000명
월간 거래액 ~8억원 ~40억원
연간 수수료 수익 ~1억원 ~5.3억원

5.3.3 Phase 2 핵심 과제

① 스케일업 기술 인프라

  • AWS GovCloud KR 기반 오토스케일링 환경 구축
  • 대용량 트래픽 처리 (동시 접속 10만명 대비)
  • 국방부 보안 감사 대응 자동화

② 제휴 금융 상품 확대

  • 나라사랑카드 4개 은행 중 2개 이상 공식 파트너 협약
  • 장병 특화 소액 보험 상품 연계 (사업중단보험 군인 버전)
  • 전역 후 청년 금융 상품 연계 (주거·창업 대출 연결)

③ 콘텐츠 생태계 강화

  • 금감원 공인 금융 교육 콘텐츠 30개 이상 탑재
  • 선배 장병 성공 스토리 (UGC) 플랫폼 구성
  • 부대별 저축 랭킹 시스템 고도화

5.4 Phase 3: 전군 제도화 — 국방부 공식 복지 프로그램 등재

5.4.1 제도화의 의미

Phase 3의 목표는 아임인이 단순한 서비스가 아니라 국방부 공식 복지 인프라의 일부 가 되는 것입니다. 이는 나라사랑카드처럼 입대 시 자동 소개되고, 장병 복지 교육의 표준 커리큘럼에 포함되는 수준을 의미합니다.

제도화 단계 내용 타임라인
국방부 공식 복지 등재 장병 복지 포털 내 아임인 공식 소개 M+24
나라사랑포털 통합 나라사랑포털 내 아임인 원클릭 가입 M+30
금융 교육 표준화 장병 금융 교육 커리큘럼에 아임인 포함 M+30
전군 적용 육·해·공군·해병대 전 장병 이용 가능 M+36

5.4.2 Phase 3 예상 규모

지표 Phase 2 달성 Phase 3 목표
적용 장병 수 ~50,000명 ~180,000명 (전체의 50%)
월간 거래액 ~40억원 ~180억원
연간 수수료 수익 ~5.3억원 ~23.8억원
아임인 내 군 비중 — 전체 거래액의 약 15%

5.5 군 특화 프로덕트 기획: UI/UX 상세 설계

5.5.1 온보딩 플로우 (입대 ~ 첫 스테이지 참여까지)

[입대]
  │
[입대 후 2주차: 기초군사훈련 수료]
  │
[나라사랑카드 발급]
  ↓
[아임인 앱 설치 (QR코드 배포)]
  │
[3분 온보딩]
  ├─ 나라사랑카드 계좌 자동 연동
  ├─ 복무 기간 설정 (전역 예정일 입력)
  ├─ 첫 스테이지 선택 (미니 / 스탠다드)
  └─ 그룹 매칭 (같은 기수·부대원 우선)
  ↓
[첫 납입일 (급여일 자동 이체)]
  ↓
[저축 계급 '이병' 획득 → 앱 내 보상]

5.5.2 게이미피케이션 시스템 전체 설계

A. 저축 계급 시스템

레벨 이름 달성 조건 보상
1 이병 첫 납입 완료 환영 뱃지 + 재무 퀘스트 오픈
2 일병 3개월 연속 납입 추가 이자 +0.5% (1개월)
3 상병 스테이지 1회 완주 다음 스테이지 우선 순번 선택권
4 병장 스테이지 2회 완주 제휴 서비스 특별 할인
5 부사관 전역 전까지 0회 중도 해지 전역 기념 금 0.1g 또는 상당 금액

B. 부대 저축 랭킹

기능 설명
중대 랭킹 소속 중대 내 개인 저축 달성률 순위
대대 랭킹 대대 내 중대별 평균 저축 달성률
전군 랭킹 전체 참여 부대 중 Top 10 발표 (월간)
보상 상위 부대에 '금융 복지 우수 부대' 표창 수여 (국방부 연계)

C. 재무 퀘스트 (금융 교육 게이미피케이션)

퀘스트명 내용 보상
첫걸음 아임인 작동 원리 영상 시청 (5분) +100포인트
복리의 마법 복리 계산 인터랙티브 학습 +200포인트
빚투 탈출 빚투 사례 분석 + 퀴즈 +300포인트
재무 설계 전역 후 재무 목표 설정 +500포인트
달인의 경지 전체 퀘스트 완료 특별 뱃지 + 이자 +1%

5.6 채널 전략: 군 내부 바이럴 구조 설계

5.6.1 초기 확산 채널

채널 방법 예상 효과
간부 교육 소대장·중대장 대상 아임인 브리핑 (30분) 상부 지지 기반 확보
분대원 버디 시스템 기존 사용자가 동기 1명 초대 시 인센티브 바이럴 확산
PX 내 QR코드 부대 식당·PX에 가입 QR코드 부착 자연 노출
유튜브·SNS "병장이 군대에서 1,500만원 모은 방법" 바이럴 영상 온라인 전파
전역자 커뮤니티 에브리타임·국방일보 채널 → 성공 사례 공유 입대 예정자 유입

5.6.2 비용 효율적 마케팅 모델

아임인의 군 채널은 국방부가 공식 채널을 제공하므로, 기존 B2C 마케팅 대비 고객 획득 비용(CAC)이 획기적으로 낮습니다.

구분 일반 B2C 앱 아임인 군 채널
고객 획득 비용(CAC) 3~10만원/명 약 5,000원~1만원/명
채널 신뢰도 중간 (광고 신뢰도 한계) 높음 (국방부 공식 채널)
이탈률 첫 달 40~60% 15~20% 예상 (복무 기간 묶임)
LTV(고객 생애 가치) 약 6~12개월 활성 18~21개월 복무 기간 전체

5.8 소결: 군 GTM 성공의 3가지 열쇠

열쇠 내용
1. 국방부 공식 파트너십 선제 확보 규제 우회 시도 대신 국방부를 공동 이해관계자로 만드는 것이 핵심
2. 금감원 샌드박스 사전 지정 법적 불확실성 제거 → 국방부의 신뢰 확보의 전제 조건
3. 장병 자발적 참여 유도 강제가 아닌 게이미피케이션과 실질적 인센티브로 자발적 가입 유도

다음 챕터에서는 유사한 민관 협력 사례를 분석하고 아임인이 벤치마킹해야 할 성공 요인을 도출합니다.
