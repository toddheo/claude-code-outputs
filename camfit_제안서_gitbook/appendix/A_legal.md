# 부록 A. 법적·제도적 검토

**[2026-05-13]**

---

## A.1 아임인의 법적 성격

아임인 스테이지는 **이용자들 상호 간에 맺어지는 '조합(Partnership) 계약' 성격의 순번계**입니다. (주)티웨이브는 이용자들로부터 자금을 수취하는 것이 아니라, 단순히 모인 계금을 순번에 맞게 지급하는 플랫폼 차원의 **'사무 관리 및 집행(Administration)'** 역할을 수행합니다.

---

## A.2 유사수신행위 비해당 공식 검토

| 검토 기관 | 검토 시기 | 결론 |
| --------- | :-------: | ---- |
| 법무법인 태평양 | 2023년 | 유사수신행위 비해당 |
| 법무법인 태평양 | 2025년 | 온라인투자연계금융(P2P) 비해당, 동일 결론 재확인 |

**비해당 근거:**
1. 유사수신행위가 성립하려면 '플랫폼이 불특정 다수로부터 출자금 명목으로 자금을 조달'해야 합니다. 아임인은 유저들 간의 상호 계약이므로 해당 없음.
2. '원금 및 확정 수익 보장' 약정이 아임인에는 존재하지 않음.
3. 안심서비스는 연체 시 채권 양도(NPL 매입) 구조로, 유사수신 리스크와 무관함.

## A.3 캠핏 파트너십 관련 법적 고려사항

| 법령 | 적용 내용 | 검토 결과 |
| ---- | --------- | :-------: |
| **전자금융거래법** | 아임인 플랫폼 운영 — 전자지급결제대행업 등록 기반 | ✅ 준수 |
| **개인정보보호법** | 캠핏 유저 정보 처리 시 동의 취득 필수 | ✅ 준수 (기존 프로세스 유지) |
| **신용정보법** | KCB 신용 조회 — 신용정보 이용·보호에 관한 법률 준수 | ✅ 준수 |
| **표시·광고 공정화법** | 캠핑 지원금 지급 조건 명확 고지 필요 | ✅ 대응 가능 |

## A.4 파트너십 협약(MOU) 주요 항목 (안)

| 항목 | 내용 |
| ---- | ---- |
| **목적** | 캠퍼 유저 공동 저축 활성화 및 상호 트래픽 시너지 창출 |
| **역할 분담** | 캠핏: 채널 제공 / 티웨이브: 플랫폼 운영 및 지원금 집행 |
| **비용 부담** | 티웨이브 전액 부담 (지원금 포함) |
| **데이터 공유** | 참여 그룹 집계 데이터 — 개인 식별 정보 제외, 동의 기반 |
| **계약 기간** | 1년 (파일럿 기간 포함), 성과 기반 갱신 |
| **해지 조건** | 양사 합의 또는 중대한 계약 위반 시 |

> 법적 절차 소요 예상 기간: MOU 체결 약 2~3주. 법무 검토·서명·운영 준비 포함.
