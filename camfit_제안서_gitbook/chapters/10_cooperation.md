# 4.1 기관 간 협력 체계 및 역할 분담

**[2026-05-13]**

---

## 파트너십 구조 개요

캠핏 × 아임인 파트너십은 **2단계 순차 확장 모델**로 설계되었습니다. 1단계에서 시스템 연동 없이 트래픽 교환의 효과를 검증하고, 그 결과를 바탕으로 2단계 화이트라벨링으로 확장합니다.

---

## 1단계: 바터(Barter) 트래픽 교환 제휴

### 역할 분담

| 역할 | 캠핏 (넥스트에디션) | 티웨이브 (아임인) |
| ---- | ----------------- | ---------------- |
| **채널 제공** | 앱/웹 내 아임인 제휴 배너 노출 지면 제공 | 아임인 앱/웹 내 캠핏 제휴 배너 노출 지면 제공 |
| **소재 제작** | 캠핏 배너 소재 제작 (자체) | 아임인 배너 소재 제작 (자체) |
| **혜택 제공** | 아임인→캠핏 유입 시 캠핏 할인 쿠폰 지급 | 캠핏→아임인 유입·가입 전환 시 4~5만 원 전용 쿠폰 지급 |
| **비용 부담** | **0원** | 전용 쿠폰 지급 비용 (티웨이브 전액 부담) |
| **실행 기간** | 협약 후 2~3주 내 가능 | 협약 후 2~3주 내 가능 |

### 캠핑 목돈 챌린지 프로모션 설계

| 항목 | 내용 |
| ---- | ---- |
| **프로모션명** | 캠핑 목돈 챌린지 |
| **대상** | 캠핏 앱을 통해 아임인에 신규 가입한 유저 |
| **혜택** | 아임인 스테이지 첫 완주 시 4~5만 원 전용 쿠폰 지급 |
| **쿠폰 용도** | 아임인 서비스 내 캠핏 예약·커머스 연계 가능 |
| **비용 부담** | 티웨이브 전액 부담 (캠핏 비용 0원) |

## 2단계: 화이트라벨링 — '캠핑 자금 모으기' 모듈 이식

### 역할 분담

| 역할 | 캠핏 (넥스트에디션) | 티웨이브 (아임인) |
| ---- | ----------------- | ---------------- |
| **플랫폼 제공** | 앱/웹 내 '캠핑 자금 모으기' 모듈 지면 제공 | 아임인 ROSCA 금융 시스템 백엔드 이식 |
| **브랜딩** | 캠핏 자체 브랜드로 서비스 운영 | 'Powered by imin' 표기 협의 가능 |
| **운영·CS** | — | 전담 (티웨이브) |
| **지원금 지급** | — | 스테이지 완주 그룹에 인당 5~7만 원 지급 (전액 부담) |
| **데이터 공유** | 참여 그룹 집계 데이터 (개인정보 제외, 동의 기반) | 전환율·리텐션 데이터 공유 |
| **비용 부담** | **0원** | 개발·운영·지원금 전액 |

### 5인 그룹 지원금 예시

| 그룹 구성 | 지원금(인당) | 총 지원금 |
| --------- | :----------: | :-------: |
| 5인 스테이지 완주 | 5만 원 | 25만 원 |
| 7인 스테이지 완주 | 6만 원 | 42만 원 |
| 9인 스테이지 완주 | 7만 원 | 63만 원 |

> **캠핏 입장에서는 비용 0원으로 유저에게 최대 63만 원의 실질 혜택을 제공하는 셈입니다.**

## 수익 구조

| 항목 | 캠핏 | 티웨이브 |
| ---- | ---- | -------- |
| **직접 수익** | 지원금 받은 유저의 캠핏 예약·커머스 전환 매출 증가 | 스테이지 수수료(1.1%) 수취 |
| **간접 수익** | 비수기 MAU 유지로 광고 수익 기반 강화 | 신규 AU 확보, LTV 누적 |
| **비용** | **0원** | 개발·운영·지원금 |
