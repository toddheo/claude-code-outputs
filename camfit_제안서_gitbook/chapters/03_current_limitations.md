# 1.3 현행 대안의 구조적 한계

**[2026-05-13]**

---

## 현재 캠퍼들이 사용하는 공동 자금 마련 방식과 그 한계

캠핑을 위한 공동 자금 마련 방법으로 현재 사용되는 대안들은 근본적인 구조적 한계를 가지고 있습니다.

---

### 대안 1. 당일 더치페이 / 카카오페이 실시간 정산

| 항목 | 내용 |
| ---- | ---- |
| **방식** | 캠핑 당일 각자 부담, 귀가 후 앱으로 정산 |
| **한계 ①** | 정산 지연·누락으로 그룹 내 갈등 발생 |
| **한계 ②** | 캠핑 당일 현금 흐름 부담 집중 |
| **한계 ③** | 큰 금액(프리미엄 사이트 등) 당일 처리 어려움 |
| **아임인 차이** | 캠핑 전에 미리 목돈을 마련해 갈등 원천 차단 |

---

### 대안 2. 개인 적금 / 은행 자유 저축

| 항목 | 내용 |
| ---- | ---- |
| **방식** | 각자 은행 적금으로 개별 저축 후 합산 |
| **한계 ①** | 그룹 공동 목표가 없어 중도 해지율 높음 |
| **한계 ②** | 저축 기간과 캠핑 타이밍의 불일치 |
| **한계 ③** | 소셜 동기부여 없음 — 혼자 모으는 건 쉽게 포기 |
| **아임인 차이** | 그룹 전체가 함께 납입 → 소셜 압박으로 완주율 극대화 |

---

### 대안 3. 개인 계좌 공동 관리 (단체 계좌)

| 항목 | 내용 |
| ---- | ---- |
| **방식** | 한 명 명의로 계좌 만들어 전체 자금 관리 |
| **한계 ①** | 계좌 관리자에 대한 신뢰 리스크 |
| **한계 ②** | 입출금 내역 공유 불투명 |
| **한계 ③** | 법적 분쟁 발생 시 보호 장치 없음 |
| **아임인 차이** | 플랫폼이 에스크로 역할, 순번 기반 투명한 자금 흐름 |

---

### 대안 4. 기존 캠핏 쿠폰·포인트 리워드

| 항목 | 내용 |
| ---- | ---- |
| **방식** | 캠핏 내 적립 포인트, 제휴 쿠폰 사용 |
| **한계 ①** | 소액 할인(수천 원~수만 원 수준)에 그침 |
| **한계 ②** | 목적성 저축 수단이 아닌 단순 할인 |
| **한계 ③** | 유저 앱 재방문 동기로 활용 불가 |
| **아임인 차이** | 인당 5~7만 원 파격 지원금 — 실질적 경제적 혜택 제공 |
