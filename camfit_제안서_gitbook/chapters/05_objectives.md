# 2.1 도입 목적 및 전략적 포지셔닝

**[2026-05-13]**

---

## 파트너십 도입의 3대 목적

### 목적 1. 비수기 리텐션 구조화 — 365일 앱을 열어야 할 이유 제공

현재 캠핏의 성수기·비수기 MAU 격차를 해소하기 위해서는 **캠핑 시즌과 무관하게 앱 방문을 유도하는 구조적 장치**가 필요합니다. 아임인의 스테이지(5~13개월)는 매월 정해진 납입일에 유저가 반드시 앱을 방문하는 동기를 내재화합니다. 특히 화이트라벨링 2단계 적용 시, 캠핑을 준비 중인 그룹이 저축 기간 내내 캠핏 앱을 중심으로 활동하게 됩니다.

### 목적 2. 목적성 소비 가속화 — 모은 돈은 캠핏에서 쓴다

아임인을 통해 목돈을 마련한 유저는 **캠핑 자금이 준비된 상태에서 캠핏으로 돌아옵니다.** 저축 완료 = 캠핑 예약이라는 자연스러운 소비 연결 고리가 형성되어, 수령 직후 캠핏 예약·커머스·체험·레저 서비스의 실질 전환율이 높아집니다.

### 목적 3. 신규 유저 교차 확보 — 양사 60만+ 유저 풀 공유

- **캠핏 → 아임인**: 캠핑 자금을 모으고 싶은 캠퍼 그룹 → 아임인 신규 가입
- **아임인 → 캠핏**: 목돈을 준비 중인 아임인 유저 → 캠핏 신규 유입
- 양방향 트래픽 교환으로 양사 모두 CAC(유저 획득 비용) 절감

---

## 전략적 포지셔닝

```
기존 캠핏 포지션:
[예약] → [커머스] → [체험/레저]
캠핑의 '실행' 단계만 커버

파트너십 이후 포지션:
[자금 마련] → [예약] → [커머스] → [체험/레저]
캠핑의 '준비~실행' 전 여정 커버
```

아임인과의 파트너십을 통해 캠핏은 **'캠핑을 예약하는 플랫폼'에서 '캠핑 여정 전체를 함께하는 슈퍼앱'** 으로 포지션을 확장할 수 있습니다.

## 단계별 전략 로드맵 요약

| 단계 | 모델 | 핵심 목표 |
| :--: | ---- | --------- |
| **1단계** | 바터 제휴 (배너 교환) | 유저 교차 유입 테스트, 전환율 검증 |
| **2단계** | 화이트라벨링 모듈 이식 | 비수기 리텐션 내재화, 캠핑 지원금 제공 |
| **3단계** | 공동 마케팅 고도화 | 캠핑 시즌 연계 프로모션, 유저 데이터 분석 |
