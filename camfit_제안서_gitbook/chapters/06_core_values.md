# 2.2 핵심 제공 가치: 안전성·유동성·소셜 동기부여

**[2026-05-13]**

## 아임인의 3축 핵심 가치

### 축 1. 안전성 — 8년간 검증된 제1금융권 수준의 리스크 관리

캠퍼들이 소그룹으로 함께 돈을 모을 때 가장 우려하는 것은 **자금의 안전**입니다. 아임인은 이 우려를 데이터로 해소합니다.

| 안전성 지표 | 수치 | 의미 |
| ----------- | ---- | ---- |
| **연체율** | **0.22%** | 국내 제1금융권 평균 대비 낮은 수준 |
| **누적 거래액** | **3조 863억 원** | 8년간 단 한 건의 플랫폼 분쟁 없음 |
| **KCB 신용평가 연동** | — | 가입 단계에서 신용 검증 완료 |
| **안심서비스** | — | 연체 발생 시 채권 양도(NPL 매입) 구조로 기존 그룹 보호 |
| **정산 파트너** | 신한은행, 케이뱅크 | 제도권 금융기관 에스크로 구조 |

> 캠퍼 그룹 내 누군가 연체하더라도 안심서비스가 작동해 나머지 멤버의 자금 흐름을 보호합니다.

### 축 2. 유동성 — 캠핑 시즌에 맞춰 목돈 수령

아임인의 스테이지는 참여자가 **원하는 순번을 선택**할 수 있습니다. 가을 캠핑 성수기에 맞춰 자금이 필요하다면 8~9월 수령 순번을 선택하고, 내년 봄 캠핑을 목표로 한다면 그에 맞는 순번을 선택합니다.

| 스테이지 옵션 | 기간 | 적합 시나리오 |
| ------------ | :--: | ------------- |
| 5인 스테이지 | 5개월 | 올해 여름·가을 캠핑 자금 단기 마련 |
| 7인 스테이지 | 7개월 | 내년 봄 성수기 목돈 준비 |
| 9인 스테이지 | 9개월 | 고가 장비 구입·프리미엄 캠핑 |
| 11인 스테이지 | 11개월 | 연간 캠핑 계획 + 여행비 통합 적립 |
| 13인 스테이지 | 13개월 | 장기 캠핑카·카라반 구매 목적 |

> **납입 범위: 월 10만 원 ~ 최대 160만 원** — 소규모 자금부터 대형 프로젝트까지 유연하게 설계 가능

---

### 축 3. 소셜 동기부여 — 함께 모으면 포기하지 않는다

혼자 모으는 적금은 쉽게 해지됩니다. 하지만 **그룹이 함께 납입하는 구조**는 완전히 다른 심리적 메커니즘을 작동시킵니다.

- **상호 책임감**: 내가 납입을 빠트리면 다른 멤버에게 피해가 간다
- **공동 목표**: "우리 가을에 캠핑 간다"는 구체적 목표가 저축 동기를 강화
- **소셜 확인**: 그룹 내 납입 현황 공유 → 자연스러운 바이럴 효과
- **완주 보상**: 목표 달성 후 아임인 지원금 최대 7만 원 — 캠핑 비용 실질 절감

> 아임인의 누적 스테이지 완주율은 업계 최고 수준입니다. 소셜 구조가 이탈을 막는 핵심 장치입니다.
