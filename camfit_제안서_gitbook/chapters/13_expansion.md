## 화이트라벨링 확대 시나리오

### 1년차 (2027년): 캠핑 목적 저축 모듈 정착

- **대상 유저**: 캠핏 MAU 100만 명 중 목적성 저축 관심 유저 예상 5~10%
- **목표 스테이지 그룹**: 500~1,000개 그룹 (2,500~5,000명)
- **예상 연간 저축 총액**: 약 150억~300억 원
- **아임인 지원금 총액**: 약 1.5억~3억 원 (티웨이브 부담)

### 3년차 (2029년): 캠핏의 '금융 버티컬' 포지션 확립

- **목표 스테이지 그룹**: 5,000개+ 그룹 (25,000명+)
- **캠핏 내 상시 프로모션**: 봄·여름·가을 시즌별 캠핑 챌린지 연계
- **B2B 확장**: 캠핏 기업 고객(팀빌딩·야외 회식) 법인 저축 모듈 적용

## 지속 운영 체계

### 연간 공동 마케팅 캘린더

| 시즌 | 프로모션 | 내용 |
| ---- | -------- | ---- |
| **봄 (3~5월)** | 봄 캠핑 자금 챌린지 | 5인 스테이지 집중 프로모션 |
| **여름 (6~8월)** | 여름 캠프 목돈 모으기 | 7인 스테이지 집중 |
| **가을 (9~10월)** | 가을 성수기 캠핑 목표 | 신규 스테이지 모집 |
| **비수기 (11~2월)** | 내년 봄 미리 준비 | 비수기 앱 재방문 유도 핵심 |

### 운영 거버넌스

| 항목 | 내용 |
| ---- | ---- |
| **월간 성과 공유** | KPI 현황 공동 검토 (비공개 데이터는 집계치만 공유) |
| **분기 리뷰** | 지원금 지급 기준·쿠폰 조건 재협의 |
| **연간 재계약** | 파일럿 이후 1년 단위 성과 기반 갱신 |
| **해지 조건** | 양사 합의 또는 중대한 계약 위반 시 |

## 장기 비전: 캠핏 × 아임인의 시너지 확장

```
현재 → 바터 제휴 → 화이트라벨링 → 공동 금융 상품 개발 → 아웃도어 슈퍼앱 금융 버티컬
```

- 캠핑 장비 구입 분할 납입 모델
- 캠핏 커머스 연계 소비 자동화 (저축 완료 = 캠핏 쿠폰 자동 발급)
- 캠핏 B2B 기업 고객 대상 단체 저축 법인 솔루션
