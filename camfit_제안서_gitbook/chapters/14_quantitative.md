## 캠핏 입장의 ROI 분석

| 항목 | 내용 |
| ---- | ---- |
| **캠핏 투입 비용** | **0원** |
| **유저 리텐션 효과** | 저축 기간(5~13개월) 동안 비수기 MAU 유지 |
| **예약·커머스 전환** | 목돈 수령 후 캠핑 예약·용품 구매 전환율 상승 기대 |
| **신규 유저 유입** | 아임인 60만+ 유저 풀에서 잠재 캠퍼 유입 |
| **브랜드 가치** | '캠핑 자금까지 해결하는 플랫폼' 차별화 포지션 선점 |

> **종합 판단:** 캠핏은 비용 0원으로 유저 리텐션·신규 유입·브랜드 차별화를 동시에 얻습니다.
