**[2026-05-13]**

## 아임인(imin)이란?

아임인은 디지털 ROSCA(Rotating Savings and Credit Association) 플랫폼입니다. 한국의 전통 금융 방식인 '계(契)'를 IT로 구현하여, 정해진 인원이 매월 일정 금액을 납입하고 순번에 따라 목돈을 수령하는 구조입니다.

### 핵심 수치 (2026년 4월 기준)

| 지표 | 수치 |
| ---- | ---- |
| 누적 거래액 | **3조 863억 원** |

| 연체율 | **0.22%** (제1금융권 수준) |
| 서비스 연수 | **8년 이상** (2016년~) |
| 연간 매출 | **116억 원** (2025년) |

## 스테이지(Stage) 운용 원리

**캠핑 목적 5인 스테이지 예시** (월 20만 원, 5개월):

```
참여자 5명이 매월 20만 원씩 납입
→ 매월 100만 원의 목돈이 순번대로 한 명에게 지급

[1개월차] A 수령: 100만 원 (앞 순번, 이자 비용 부담)
[2개월차] B 수령: 100만 원
[3개월차] C 수령: 100만 원 (중간 순번, 이자 없음)
[4개월차] D 수령: 100만 원
[5개월차] E 수령: 100만 원 + 이자 수익 (뒷 순번 보상)
```

> 5명 모두 동일한 100만 원을 수령하지만, **앞 순번은 먼저 쓸 수 있는 유동성**, **뒷 순번은 이자 수익**이라는 다른 혜택을 받습니다.

## 캠핏 파트너십 적용 시 운용 방식

### 1단계 (바터 제휴): 독립 서비스로 아임인 앱에서 이용

- 캠핏 유저가 배너 클릭 → 아임인 앱 설치 → 캠핑 목적 스테이지 생성/참여
- 스테이지 완주 시 아임인 캠핑 전용 쿠폰(4~5만 원) 지급

### 2단계 (화이트라벨링): 캠핏 앱 내 '캠핑 자금 모으기' 모듈

- 캠핏 앱 내에서 그룹 저축 기능을 캠핏 브랜드로 이용
- 아임인의 금융 인프라(KCB 연동·안심서비스·정산 시스템)는 백엔드에서 운영
- 'Powered by imin' 표기 여부는 협의 가능
- 스테이지 완주 시 아임인 캠핑 지원금(인당 5~7만 원) 지급 — 티웨이브 전액 부담
