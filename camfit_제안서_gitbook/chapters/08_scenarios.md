**[2026-05-13]**

---

## 시나리오 A. 가족 캠핑형 — 연간 캠핑 자금을 미리 모은다

**상황:** 부부 + 성인 자녀 2명(19세 이상), 연 2~3회 캠핑. 캠핑 비용이 매번 부담스럽고, 사이트비·식재료·용품 구매를 하나씩 처리하다 보면 지출 관리가 어렵다.

| 항목 | 내용 |
| ---- | ---- |
| **스테이지 구성** | 5인 스테이지, 월 20만 원, 5개월 |
| **수령액** | 100만 원 |
| **아임인 캠핑 지원금** | +5만 원 |
| **실질 캠핑 자금** | **105만 원** |
| **적합 용도** | 가을 성수기 사이트비 + 캠핏 커머스 식재료 선주문 |

> "5개월 동안 매월 20만 원씩 모아서 캠핑 다녀오고 5만 원도 더 받았어요."
