## 파일럿 운영 프로세스

### Step 1. 협약 체결 (D-Day)

- 제휴 협약서(또는 MOU) 서명
- 배너 노출 지면, 혜택 조건, KPI 공유 방식 합의

### Step 2. 배너 소재 제작 (D+1주~2주)

- **캠핏 측**: 아임인 제휴 배너 소재 제작 (아임인 BI 가이드 제공)
- **티웨이브 측**: 캠핏 제휴 배너 소재 제작 (캠핏 BI 가이드 기반)
- 양사 상호 검수 및 승인

### Step 3. 배너 노출 시작 (D+3주)

- 캠핏 앱 메인/예약 페이지 내 아임인 제휴 배너 노출
- 아임인 앱 홈/스테이지 페이지 내 캠핏 제휴 배너 노출
- 유입 트래킹 링크 설정 (UTM 파라미터 등)

### Step 4. 중간 점검 (D+6주, 7월 중)

- 클릭률(CTR), 설치율, 가입 전환율 공유
- 성과 미달 시 배너 위치·소재 A/B 테스트 실시

### Step 5. 파일럿 결과 분석 (D+12주, 8월 말)

- 최종 전환율·완주율 분석 보고서 공유
- Phase 2 (화이트라벨링) 진행 여부 공동 결정

## 파일럿 KPI

| 지표 | 목표치 | 측정 방법 |
| ---- | :----: | --------- |
| 캠핏→아임인 배너 클릭 | 1만 회+ | UTM 트래킹 |
| 아임인 신규 가입 (캠핏 유입) | 500명+ | 아임인 내부 집계 |
| 스테이지 참여율 (가입 대비) | 30%+ | 아임인 내부 집계 |
| 아임인→캠핏 배너 클릭 | 5,000회+ | UTM 트래킹 |
| 캠핏 신규 유입 (아임인 유입) | 300명+ | 캠핏 내부 집계 |

## 파일럿 성공 기준 및 Phase 2 전환 조건

파일럿 종료 후 아래 기준 2개 이상 충족 시 Phase 2(화이트라벨링)로 전환 권고:

1. **아임인 신규 가입 300명 이상** 달성
2. **스테이지 참여율 25% 이상** (가입자 중 스테이지를 개설하거나 참여한 비율)
3. **캠핏 신규 유입 200명 이상** 달성
4. **배너 CTR 1.5% 이상** 유지
